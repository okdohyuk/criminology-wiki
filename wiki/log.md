# 작업 이력 (Log)

위키에서 일어난 모든 주요 작업을 시간순으로 기록한다. 새 항목은 **맨 위**에 추가.

## 로그 포맷

```
### YYYY-MM-DD — {작업 유형}

- **작업**: Ingest / Update / Health Check / Query / Delete
- **원본**: raw/파일명 (해당 시)
- **변경**: 생성/수정/삭제된 페이지
- **요약**: 무엇을 어떻게 정리했는지 1~3줄
```
