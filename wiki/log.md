## 로그 포맷

```
### YYYY-MM-DD — {작업 유형}

- **작업**: Ingest / Update / Health Check / Query / Delete
- **원본**: raw/파일명 (해당 시)
- **변경**: 생성/수정/삭제된 페이지
- **요약**: 무엇을 어떻게 정리했는지 1~3줄
```
