# 가설 (Hypothesis)

> **학습 takeaway**: 가설은 **"A라면 B일 것이다"**의 형태를 가진 **검증 가능한 진술**. 단순 설명문이나 기대가 아니다. "청소년 비행수준은 높다"는 가설이 아니고, "부모 감독이 낮을수록 비행이 증가할 것이다"가 가설이다. 이 차이를 정확히 잡는 것이 연구설계의 기본기.

## 정의

두 개 이상의 변수 또는 현상 간의 관계를 **검증 가능한 형태**로 제시한 가정적 진술.

## 기본 형식

```
만약 A라면 B일 것이다
A가 높을수록 B는 증가/감소할 것이다
```

- 단순 설명문 ❌
- 상관관계 진술 (방향 불명확) △
- **인과 방향이 분명한 진술** ✅

## 좋은 가설의 5대 기준

| 기준 | 의미 |
|------|------|
| **명료성** | 개념이 분명해야 함 |
| **간결성** | 불필요하게 복잡하지 않을 것 |
| **검증 가능성** | 경험적으로 참·거짓 확인 가능 |
| **이론 연관성** | 연구문제·선행연구에 의해 뒷받침 |
| **양화 가능성** | 변수 측정이 가능 |

## 예시 비교

| ❌ 나쁜 가설 | ✅ 좋은 가설 |
|-------------|-------------|
| 경찰이 친절하면 신뢰가 높아진다 | 절차적 공정성 인식수준이 높을수록 경찰신뢰수준은 증가할 것이다 |
| 인터넷 사용시간과 사이버불링은 관련이 있다 | 인터넷 사용시간이 많을수록 사이버불링 가해 수준은 증가할 것이다 |
| 청소년 비행수준은 높다 | 부모의 감독수준이 낮을 경우 청소년 비행수준은 증가할 것이다 |

## 가설과 명제와 이론

| 수준 | 의미 |
|------|------|
| **가설** | 변수 관계에 대한 검증 가능한 진술 |
| **명제** | 반복 검증으로 지지된 진술 |
| **이론** | 여러 명제가 연결된 설명 체계 |
