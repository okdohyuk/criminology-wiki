# 가정폭력 (Domestic Violence)

> **학습 takeaway**: 가정폭력은 소득·교육·인종과 무관하게 발생하는 구조적 현상이다. "부부싸움은 칼로 물 베기"라는 통념 자체가 피해자를 가두는 이데올로기다. **폭력의 순환이론**(긴장→폭발→사과→긴장)과 **매 맞는 아내 증후군**은 피해자가 떠나지 못하는 이유를 심리적으로 설명하고, **권력과 통제의 바퀴이론**(페미니스트 이론)은 구조적으로 설명한다. 한국 형법 강간죄의 변화(1970→2009→2013)가 이 이론들의 실정법화 과정을 보여준다.

## 정의

**가정 내 혹은 가족 구성원 사이에서 일어나는 폭력.** 신체적 폭력뿐 아니라 정서적·경제적·성적 폭력 포함.

> 가정폭력은 소득·교육수준, 지역, 인종과 무관하게 발생한다.

## 주요 유형

| 분류기준 | 유형 |
|---------|------|
| **폭력 형태** | 신체적, 정서적, 경제적, 방임 |
| **피해 대상** | 아내, 남편, 자녀 → [[아동학대]], [[노인학대]] |
| **가해자 유형** | Saunders: 공격적/일반적/감정적 폭력성향; Dutton: 정신병리적/과도하게 통제된/순환적 불안정; Holtzworth-Munroe: 가정 내/일반화반사회적/우울경계선 |

## 특별유형

### 배우자 구타 (Spousal Battering)

학대자가 **권력과 통제**를 유지하기 위해 배우자에게 가하는 학대. 19세기 "엄지의 법칙"(엄지 두께보다 얇은 회초리는 합법)에서 볼 수 있듯 오랫동안 사회·법률적으로 허용되어 왔다.

**주요 원인** (Harris & Dewdney, 1994): 남성의 과음·질투·낮은 자존감·정서불안, 경제문제·스트레스, 여성에 대한 배려부족, 문제해결 수단으로서 폭력 수용.

**O.J. Simpson 사건**: 가정폭력의 실제 진행과정을 보여주는 전형적 사례. 미식축구 선수 O.J. 심슨이 배우자를 살해한 혐의로 기소되었으나 형사재판 끝에 무죄 판결을 받은 사건(1990년대). 가정폭력의 심각성과 법적 처벌의 어려움을 보여주는 상징적 사건.

### 부부강간 (Marital Rape)

| 연도 | 한국 판례 변화 |
|------|-------------|
| 1970 | 실질적 부부관계 유지 중 강간 불성립 |
| 2009 | 이혼 의사 합치 상태의 강간 처벌 가능 |
| 2013 | **정상적 부부관계라도 강제 성관계 = 강간죄** |

> 2012년 형법 개정: "부녀" → "사람"으로 개정 → 여성의 성적 자기결정권 강화

### 배우자 살해 (Spousal Homicide)

반복되는 최악의 가정폭력에 대한 정당방위로 나타나기도 함 → **매 맞는 아내 증후군**의 극단적 결과.

## 이론적 설명

### 사회심리학적 이론

| 이론 | 핵심 |
|------|------|
| **일반체계이론** | 폭력주기를 안정·항상성의 시스템으로 봄 |
| **폭력의 하위문화이론** | 폭력을 정당화하는 특정 하위문화 가치체계 |
| **사회학습이론** | 폭력행동은 관찰·학습의 산물 |
| **자원이론** | 가족통제 자원이 없는 가장이 폭력을 자원으로 활용 |
| **갈등이론** | 가족구성원 갈등관리 실패 → 폭력적 방법 사용 |
| **교환/사회통제이론** | 보상·손실 교환 중단으로 폭력의 지속·중단 설명 |
| **학습된 무기력이론** | 피해자가 폭력에 노출된 무기력한 상태에 익숙해짐 |

### 사회문화적 이론

**구조적 이론**: 스트레스·좌절·박탈의 차별적 분포 → 빈곤층·인구밀집지역 폭력 빈발.

**페미니스트 이론 — 권력과 통제의 바퀴**: 가부장적 구조에서 남성이 여성을 통제·억압하는 **8가지 전략**:

| 전략 | 내용 |
|------|------|
| 위협 이용 | 표정·행동으로 겁주기, 물건 부수기, 무기 꺼내놓기 |
| 정서적 학대 | 모욕, 죄책감, 스스로 미쳤다고 생각하게 하기 |
| 고립 이용 | 행동 제한, 외부활동 통제, 질투심 활용 |
| 축소·부정·비난 | 학대 없었던 것으로 하기, 피해자 잘못 비난 |
| 자녀 이용 | 자녀 통해 메시지 전달, 데려가겠다고 협박 |
| 남성특권 이용 | 하인취급, 독단적 결정, 남녀역할 고정 |
| 경제적 학대 | 취업 방해, 생활비 제약, 금전 강취 |
| 강요와 협박 | 자해·자살 협박, 고소취하 종용 |

### 폭력의 대물림이론

아동기 피학대 경험 → 성인기 폭력 성향. 그러나 대다수 피학대 아동이 폭력적 성인으로 자라는 것은 아님 — 폭력의 대물림을 끊을 수 있는 단초.

**대물림 끊기 — EBS "아버지 학교" 사례**: 어린 시절 아버지로부터 받은 상처를 의식적으로 직면하는 과정. 핵심 메커니즘:
1. **메타인지**: 자신이 어린 시절 경험을 자녀에게 반복하고 있다는 것을 인식
2. **멈춤**: 분노·두려움·수치심이 올라오는 순간 정서적 분리
3. **재구성**: 폭력적 부모와의 동일시를 해체하고 새로운 관계 패턴 학습

> **폭력의 순환 ↔ 연쇄살인 욕구 사이클**: 가정폭력의 긴장→폭발→사과→평온→긴장 순환 구조는 연쇄살인범의 욕구 사이클(충동 고조→행동→잠시 안도→다시 충동)과 구조적으로 유사하다. 두 순환 모두 "내부 압력이 임계점에서 폭발하고 방출 후 일시적 완화"라는 패턴을 공유.

**피해자화의 대물림**: 대물림이론은 가해자 성향의 전수만을 설명하지 않는다.

```
아동기 가족폭력 피해/목격
  → 청소년기: 학교폭력 피해 노출 (폭력 허용도 상승)
  → 청년기: 데이트폭력 피해 (폭력도 의사소통의 수단이라는 내면화)
  → 성인기: 배우자폭력 피해 (순환 지속)
```

**폭력 허용도(Violence Tolerance)**: 반복적 폭력 노출이 "폭력도 관계의 일부"라는 인식을 형성 → 피해자가 상대방의 폭력을 수용해 버리는 심리적 기제. 가해자를 만드는 동시에 피해자를 만드는 이중 작동.

### 폭력의 순환이론 (Cycle Theory of Violence)

```
긴장 형성단계 → 감정폭발·구타단계 → 평온한 사랑의 휴지단계(사과·약속) → 긴장 형성단계 ...
```

- 사과와 약속은 근본적 해결이 아니라 순환의 일부
- 피해자는 반복되는 공포를 느낌

### 매 맞는 아내 증후군 (Battered Woman Syndrome)

1970년대 가정폭력에 시달리던 여성의 남편살해 사건에서 처음 사용.

- PTSD + 계속되는 폭력 → 벗어날 길 없다는 절망과 무기력
- 학습된 무기력 상태, 자기파괴적 대응
- DSM·ICD 진단분류로는 인정받지 못함
- 최근 판례: 사안의 급박성·수단의 비례성 인정 → **[[정당방위]] 인정 가능**
  - "폭력의 순환이론"을 법적 "현재성" 개념으로 번역하는 과정 진행 중

> 피해자가 가해자를 떠나지 못하는 이유: 보복 두려움, 경제 수단 결여, 개선 희망, 사회적 낙인 — 학대를 즐기기 때문이 아니다.

## 데이트폭력과 IPV

### Victim vs Survivor 용어

- **피해자(Victim)**: 법적·통계적 맥락에서 사용. 피해 사실을 기술하는 중립적 용어
- **생존자(Survivor)**: 트라우마를 극복하고 능동적으로 삶을 재구성하는 주체성 강조
- 임상·지원 현장에서는 당사자가 선택한 용어를 존중하는 것이 원칙 — "피해자"가 낙인처럼 느껴질 때 "생존자"가 회복 언어로 기능함

---

**IPV (Intimate Partner Violence)**: 가정폭력과 데이트폭력을 포괄하는 해외 표준 개념.
- 한국: "가정폭력"(혼인·가족 관계) vs. "데이트폭력"(연인 관계) — 법률적 구분
- 해외: 결혼 여부와 무관하게 "친밀한 파트너에 의한 폭력" 하나로 통합

**메커니즘은 동일**: 은폐성, 습관성, 강화(learning), 폭력의 순환 — 가정폭력과 데이트폭력이 공유하는 특징. "가정이라는 단어를 연인 간으로 치환하면 법이 거의 같다."

**가스라이팅(Gaslighting)**: 피해자로 하여금 자신의 판단과 인식을 의심하게 만드는 심리적 조작.
- "네가 예민한 거야", "그런 일 없었잖아"
- 죄책감을 주입시켜 피해자가 스스로의 경험을 부정하게 만드는 형태의 정서적 폭력
- 가정폭력·데이트폭력·직장 내 갑질 모두에서 발생

**디지털 가정폭력 (최신 연구 주제)**: 신체적 접촉 없이 디지털 기기로 행사하는 폭력.
- 위치 추적 (GPS·스마트폰)
- 이메일·메시지 감시
- SNS 계정 통제
- 온라인 스토킹
→ 직접 구타는 없지만 공포와 통제의 구조는 동일. 호주·뉴질랜드에서 관련 연구 활발.

## 실태 (2016 여성가족부 실태조사)

- 기혼남녀 1년간 부부폭력 발생률: **41.5%**
- 가장 많은 유형: 통제(37.7%) > 정서적 폭력(12.5%) > 신체적 폭력(3.7%)
- 원인 1위: 성격차이(45.3%)

## 법적 처리 통계와 제도

**가정폭력 입건율 16%**: 신고된 가정폭력 사건 중 실제로 입건되는 비율 — 매우 낮음.
- 구속률: 1% 미만
- 기소율: 26.7%

→ 신고 자체도 어렵고(피해자 비난, 보복 두려움), 신고해도 처벌로 이어지는 경우가 극소수.

**특별법 우선의 원칙**: 가정폭력 사범에는 형법상 폭행죄(2년 이하 징역·500만 원 이하 벌금)가 아닌 **가정폭력범죄의 처벌 등에 관한 특례법**(5년 이하 징역)이 우선 적용됨.

**두 가지 처리 경로**:
```
가정폭력 사건 발생
  → 형사사건 (형사법원): 교정 불가, 처벌이 필요한 경우
  → 보호사건 (가정법원): 교육·치료·지원으로 가정 회복 가능한 경우
```

**최근 개정**: 가정폭력 현행범 체포 가능, 접근금지 위반 최대 징역 3년.

### 경찰 개입 RCT — Sherman의 실험

**실험 설계**: 가정폭력 현장 경찰 대응 방식의 재범 억제 효과를 무작위 대조 시험(RCT)으로 검증.

| 개입 방식 | 결과 |
|---------|------|
| **강력 대응** (유치장 연행) | 재범 감소 효과 **있음** |
| **주의 조치** (경고만) | 재범 감소 효과 **없음** |

**핵심 발견 — 조절 변수로서의 직업**:

> "가해자가 **직장이 있을 때**는 강력 대응이 재범을 억제한다. 그러나 **직장이 없는 가해자**에게는 강력 대응도 효과가 없다."

**함의**: 단일 "강력 대응" 정책보다 가해자의 사회적 자본(직업·가족·지역사회 결속) 수준에 따른 **차별화된 맞춤형 전략**이 필요. 잃을 것이 있는 사람에게만 처벌 위협이 억제 효과를 낸다 → [[억제이론]]의 실증 검증이자 한계.

> **의외의 연결점** ↔ [[교도소화]] 유입 모델: 직장 없는 가해자에게 강력 대응이 효과 없는 구조는 교도소화 유입 모델의 논리와 동일 — "사회적 자본이 없는 사람은 처벌 위협에 무반응." 교정처우도 가정폭력 개입도, 개입 효과는 당사자가 가진 사회적 자원에 달려 있다.

| 기관 | 역할 |
|------|------|

| 상담소·보호시설 | 치유 상담, 피해자 보호, 임시보호·자활지원 |

## 의외의 연결점

- **폭력의 순환이론 ↔ 학습된 무기력이론**: 순환이 반복될수록 무기력이 강화된다 — 두 이론이 동일한 과정의 심리적·행동적 두 측면을 설명 → [[피해자-비난론]] (피해자가 왜 떠나지 않는지를 비난론으로 오용하는 경로)
- **페미니스트 이론 ↔ 갈등이론**: 권력과 통제의 바퀴는 갈등론의 "지배집단이 종속집단을 통제하는 구조"를 가족 내부에 적용한 것 → [[범죄패러다임|갈등론]]
- **폭력의 대물림 ↔ 사회학습이론**: 폭력이 세대를 넘어 전승되는 메커니즘이 반두라의 관찰학습과 동일 → [[하위문화이론]]
- **폭력 허용도(Violence Tolerance) ↔ [[피해자-비난론]]**: 피해자가 폭력을 허용하게 되는 내면화 과정이, 외부인의 시각에서는 "당할 짓을 했다"는 피해자 비난으로 왜곡됨 — 같은 현상의 내부(심리적) vs 외부(사회적) 해석
- **가스라이팅 ↔ [[피해자화]]**: 가스라이팅은 피해자가 자신이 피해자임을 인식하지 못하게 하는 2차 피해자화 기제 — 공식통계에서 신고율이 낮은 이유 중 하나
- **매 맞는 아내 증후군 ↔ 정당방위**: 급박성이 없는 상황(잠든 남편)에서의 살해에 정당방위를 인정하는 것은 기존 정당방위 요건(즉각적 위협)을 심리학적 개념으로 확장한 사례
- **경찰 개입 RCT(직장 없는 가해자) ↔ [[교도소화]] 유입 모델**: 직장·사회적 자본이 없는 가해자는 강력 경찰 개입에도 재범 억제 효과가 없다. 이는 교도소화 유입 모델이 말하는 "입소 전 사회적 자원 부재가 수용 후 행동 패턴을 결정한다"와 동일한 구조 — 형사사법 개입의 효과는 당사자의 사회적 자원 수준에 달려 있다
- **Victim vs Survivor 용어 ↔ [[피해자-비난론]]**: 피해자(victim) 언어는 수동성을, 생존자(survivor) 언어는 능동성을 함의한다. "피해자답지 않다"는 비난은 생존자가 능동적 대응을 했을 때 발생한다 — 두 용어의 긴장이 피해자 비난론의 구조를 보여줌
