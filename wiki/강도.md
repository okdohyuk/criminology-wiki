# 강도 (Robbery)

> **학습 takeaway**: **재산범죄 + 폭력범죄**의 이중 성격을 동시에 갖는 범죄. [[합리적-선택-이론]]의 교과서적 적용 사례이기도 하다. 강도범이 대상·장소·도주경로를 **계산**한다는 사실을 이해하면 왜 CPTED·상황적 예방이 강도에 효과적인지 한눈에 보인다.

## 정의

폭행 또는 협박으로 타인의 재물을 **강취**하거나 재산상 이익을 취하는 범죄.

## 5대 특성

강도는 다음 5가지를 동시에 가진 유일한 범죄유형이다:

| 특성 | 내용 |
|------|------|
| **재산범죄** | 금품 획득이 목적 |
| **폭력범죄** | 신체 안전을 직접 침해 |
| **도구적 흉기** | 재물 탈취를 위한 **주목적 수단** — 협박·통제 |
| **보충적 흉기** | 저항·도주 시 상해·살인의 **보조 수단**으로 전환 |
| **합리적 의사결정자** | 대상·장소·시간·도주경로를 사전 계산 |

흉기가 **도구적 기능**(협박)에서 **보충적 기능**(상해)으로 전환되는 순간이 강도가 강도살인으로 이어지는 임계점이다.

## 범행수법 4분류

| 수법 | 특징 |
|------|------|
| **침강도** | 주거·상업시설 침입 후 강도 |
| **노상강도** | 거리·공공장소에서 통행인 대상 |
| **인질강도** | 피해자를 인질로 잡고 금품 요구 |
| **중강도(Armed Robbery)** | 무장 강도 — 흉기 사용이 명시적 |

## [[합리적-선택-이론|계산하는 강도범]]

강도범은 전형적으로 다음을 저울질한다:
- 금품 규모
- 체포 위험
- 범행 용이성
- 도주 경로

구체적 행동:
- 더 약해 보이는 대상 선별
- 인적 드문 골목, CCTV 사각지대 선택
- 오토바이 등 도주수단 준비
- 훔친 휴대전화 초기화 후 처분

## 유형

- **특수강도** — 흉기 사용 또는 2인 이상 합동
- **준강도** — 절도 후 체포 면피를 위해 폭행·협박 사용
- **인질강도**
- **해상강도**
- **상습강도**

## 면식강도의 예외성

면식관계 강도는 **금전보다 보복감정**이 앞서는 경우가 많아 피해자의 신체적 피해가 더 커지기 쉽다. 일반적 계산 모델에서 벗어나는 사례.
