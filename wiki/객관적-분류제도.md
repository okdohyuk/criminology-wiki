# 객관적 분류제도 (Objective Classification System)

> **학습 takeaway**: 객관적 분류제도는 교도관의 주관적 판단 대신 **표준화된 도구**로 수형자를 분류하는 체계다. 결정적 관점 전환: 이 제도는 "인권 진보"의 산물이 아니라 **1·2차대전·대공황으로 시작된 노동력 활용 시대 + 1960~70년대 교화프로그램과 교정직 인력 부족의 충돌**이 만든 **행정공학적 도구**다. 6가지 목적(과밀수용·재무·의사결정·운영·기획·공공안전)은 병렬이 아니라 국가 관점 5개가 충족되어야 국민 관점의 공공안전이 달성되는 위계 인과 사슬을 이룬다. 세대별 재범위험성 평가도구([[재범위험성평가]])와 맞물려 1세대 비구조화 임상판단 → PCL-R 같은 1.5세대 구조화 임상판단 → 2세대 보험계리적 정적요인 → 3세대 동적요인 통합으로 진화하며, 분류는 교도소 입소부터 가석방 심사까지 교정 전 과정의 의사결정을 구조화하는 **데이터 인프라**다. 핵심 맹점은 시설환경 영향이 도구의 개인 측정을 왜곡한다는 점.

## 등장 배경

| 시대 | 교정 철학 | 시대적 메커니즘 |
|------|---------|---------|
| **19세기** | 교화·개선 중심, 사회 재통합 강조 | 임의적 배치, 일관성 없음 |
| **20세기 초** | 경제적·행정적 효율성, **재소자 노동을 통한 소득 확보** | 1·2차 세계대전·대공황으로 인한 노동력 부족 → "무료 인력"으로서의 재소자 노동 활용 (강의자 표현). 노동 능력 위주 분류 |
| **1960~70년대** | 교화개선 프로그램 등장 | 프로그램은 등장했으나 **교정직 인력 자원의 부재** → 효과적 재소자 관리를 위한 객관적 분류제도 등장의 직접 동인 |

> **핵심 인과**: 객관적 분류제도는 "교화 이념이 좋아져서" 등장한 것이 아니라, **교화프로그램과 교정직 인력 사이의 자원 격차**를 합리적으로 메우기 위한 행정공학적 도구로 등장했다. 이 출발 동기가 이후 6가지 목적 중 "재무관리·과밀수용 관리"가 큰 비중을 차지하는 이유다.

### 초기 분류 체계의 한계 3가지

1. **임의성·일관성 부족**: 분류 도구의 타당성 검증 미비 + 분류 기준 일관성 결여 → 같은 수형자를 다른 담당자가 다르게 분류
2. **분류담당자 주관성** (가장 큰 문제, 강의자 강조): 일부 분류범주·구성요소에 대해 담당자의 재량권 남용 허용. 누적된 개인 경험을 객관적 지표 대신 사용 → "얘는 당연히 이러겠구나" 식의 사전판단
3. **개별 보안수준 불일치**: 사회적 배경·전과 등에 기초해 개별적 보안수준 결정 → 위험도와 수용 시설이 맞지 않아 과잉·과소 보안 문제 발생

## 목적 6가지

객관적 분류제도가 달성하고자 하는 6가지 공식 목적 (PDF 슬라이드 6):

| 순서 | 목적 | 의미 | 관점 |
|------|------|------|------|
| **①** | 과밀수용의 관리 | 자원의 효율적 활용 / 장래 수용인구 증감의 예측 / 프로그램 기획과 계획의 도움 | 국가 |
| **②** | 재무관리 | 교정자원의 오집행 감소 / 제한적 자원의 효과적 활용 / 고비용 시설의 불필요 인원 감소 | 국가 |
| **③** | 양질의 의사결정 | 주관적 분류의 한계점(문서의 부재·비구조화·범주의 부족 등) 보완 / 공평성·일관성 지향 / 과다분류·과대감시 감소 | 국가 |
| **④** | 교도소 관리와 운영 | 개별 자소자 욕구 파악 / 적절한 보안·구금 수준 설정 / 교도관↔재소자 간 쟁송·폭력·공포·권력남용 가능성 감소 | 국가 |
| **⑤** | 교정 기획과 심사 | 예산책정 기준 확보 / 교정당국의 데이터 산출 / 적절한 교화프로그램 기획 | 국가 |
| **⑥** | 공공의 안전과 보호 | 고위험 수형자 분리 + 도주·재범 예방을 통한 지역사회 안전 보장 | **국민** |

### 목적의 인과 사슬 (강의자 정리)

6개 목적은 병렬이 아니라 **위계적 인과 사슬**로 이해해야 한다:

```
과밀수용 관리 → 재무관리 → 교도소 관리·운영 → 양질의 의사결정
   (데이터 확보)  (예산 최적화)  (현장 안정화)        (간부 판단력)
                                                     ↓
                                            교정 기획·심사 (정책 환류)
                                                     ↓
                                         최종 목적: 공공의 안전과 보호 (국민)
```

→ 국가 관점 5개 목적이 충족되어야 국민 관점의 마지막 목적(공공안전)이 달성된다. 객관적 분류제도가 **행정공학적 도구**인 이유.

## 효과적 분류제도의 조건 8가지 (PDF 정확본)

PDF 슬라이드 8~9가 제시하는 객관적 분류제도가 성립하기 위한 8가지 조건:

| # | 조건 | 의미 |
|---|------|------|
| 1 | **검증된 분류 도구** | 재소자에게 타당한 것으로 검증된 분류 도구 사용 |
| 2 | **동일한 점수화 기준 적용** | 모든 재소자에게 적용 가능한 동일 구성요소의 점수화 분류 |
| 3 | **관련 요소에 근거한 결정** | 재소자 배치와 관련된 것으로 보이는 요소의 적용에만 기초한 결정 |
| 4 | **재소자와 보안분류 간 일치** | 재소자의 배경과 일치되는 보안분류 지정 |
| 5 | **예외 최소화** | 유사한 사례에 대한 분류담당자 간 유사한 결정 (편차 최소화) |
| 6 | **재소자의 참여** | 재소자가 적극적으로 자기를 표현·답변해야 정확한 분류 가능 |
| 7 | **쉬운 이해도** | 직원·재소자 모두 분류 절차와 질문지를 이해할 수 있어야 함 |
| 8 | **체계적·효율적인 심사(monitoring)** | 지속적 모니터링 체계 마련 |

> **강의자 강조**: 6번 "재소자 참여"와 7번 "쉬운 이해도"는 한 쌍이다. 재소자가 질문지를 이해하지 못하면 답변·서류 작성이 불가능해 분류 자체가 무의미해진다. 분류는 일방적 측정이 아니라 **재소자-담당자 간 의사소통 작업**.

## 경험적 평가 (PDF 슬라이드 10)

### 긍정적 효과 3가지

1. **과다분류 감소** — 모든 수형자를 고위험으로 묶던 관행을 객관 도구가 절제
2. **의사결정의 일관성 향상** — 관리자가 상황 판단·대응의 기준을 공유
3. **유용한 예측 도구로서의 사용** — 시설 내 행동 문제 예측

### 맹점: 시설환경 영향 큼

> 분류도구는 개인 특성에 집중하지만, **교도소 환경 자체의 영향**을 무시할 수 없다. 과밀수용·폭력적 교도소 문화·특정 시설 배치가 수형자 행동을 변화시킨다 — 분류 시점에 측정한 위험도가 실제 수용 환경에서 달라질 수 있다는 구조적 한계. 강의자 표현: "재소자가 실질적으로 느끼는 거는 환경에 대한 게 굉장히 크기 때문에 환경 영향은 절대 무시할 수 없다."

## 세대별 평가도구와의 연결

객관적 분류제도가 사용하는 도구는 [[재범위험성평가]]의 세대별 진화와 맞물린다. 강의 복습 부분(transcript 후반)이 정리한 내용:

### 1세대 — 비구조화된 임상 판단

- **비구조화된 임상 판단**: 평가자가 매뉴얼·지침 없이 면담하면서 개인적 학문 지식·주관에 의존해 위험성 판단
- 평가자는 임상가일 수도, 교정직원·법원조사관·보호관찰관일 수도 있음
- 발전 단계: 단순 면담 → **MMPI** 등 심리검사 활용
  - MMPI-2 = 성인용 / MMPI-A = 청소년용 (한국에서는 청소년 보호관찰 단계에서 주로 활용)
  - 다면적 인성검사. 우울증·조현·강박 등 척도 조합 프로파일
  - 그러나 "어떤 성향이면 범죄를 더 많이 저지를지" 기준이 없어 예측 타당도 부족

### 1.5세대 — 구조화된 임상 판단

- **PCL-R**(사이코패스 진단도구)이 전형. 시나리오와 분기 질문으로 구성
- "기생적 생활을 하는가" 같은 동적 요인을 평가하기 위해 **표준화된 인터뷰 시나리오** 사용 → 어떤 평가자가 해도 비슷한 신뢰도·타당도
- 강의자 정의: "전문가마다 다른 정의·방식을 통제하기 위해, 질문을 순서대로 하도록 시나리오를 짠 것"

### 2세대 — 정적 요인 보험계리적 판단

- 정적 요인(static risk factor)만으로 구성: 변하지 않는 사실 (과거 유죄판결 전력·30일 구류 이상 형 집행 전력·최근 범행 당시 나이·범행 전 진행 기간·집행유예·가석방·연금·규칙 위반·탈주실패 경합 여부·마약 중독 전력 등)
- **서류만 보고도 점수 산정 가능** — 대상자가 앞에 없어도 평가
- **최초의 객관적 분류 도구는 미국 일리노이주의 가석방 신사도구** (강의자 명시)
- **보험계리적 판단(actuarial judgment)**: 보험 가입 시 가족력 통계로 보험료 산정하듯, 누적 범죄자 데이터·통계에 기반한 확률적 판단
- 점수 척도 함정: "점수가 높을수록 위험성이 낮은" 척도도 존재 — 척도마다 방향이 다름

### 3세대 이상 — 동적 요인 통합

- 동적 요인(dynamic risk factor): 개입·프로그램으로 바꿀 수 있는 변수 (반사회적 성향·태도 등 [[재범위험성평가]] Big 4 / Next 4 / Moderate 4)
- 이 단계부터 분류가 단순 분리에서 **처우 설계의 도구**로 전환

### 한국 현장 도구

- **KORI-PI**(한국교정본부 위험성-욕구 평가서): 신입 분류심사의 핵심 도구. 재심사에도 사용
- **고위험 평가서**: 신입 평가에서 고위험군으로 분류된 사람에게 별도로 따라붙는 평가서
- **RM 평가서**(Risk Management): 가석방 심사 단계의 핵심 자료. 신입 평가에 RM이 붙은 사람은 재심사·가석방까지 RM이 따라붙음
- **교도관 평상시 관찰 평가서**: KORI-PI나 RM 같은 표준도구가 아닌, 평소 교도관이 관찰한 행동 기록

## 분류심사의 실무 구조 (예규 기반)

발표자가 법무부 예규를 조사하며 정리한 분류심사 실무:

### 분류조사 방법
- 수용기록 확인
- 본인 상담
- 가족 면담
- 관계기관 조회

### 분류초 심사표 구성
1. 인적사항
2. 기본 상황
3. 심사 위초 등급내역
4. 종류별 등급 기재
5. 분류 심사 거부 사유 기재란

### 분류심사 거부권

수형자는 **분류 심사를 거부할 수 있다**(법무부 예규). 발표자도 자료 조사 중 처음 알게 된 사실이라고 언급. 거부 시 불이익(프로그램 배정 불가, 처우 불이익)을 감수해야 하는 구조적 압력이 존재 — 형식적으로는 권리, 실질적으로는 거의 행사되지 않음.

## 가석방 심사와의 연결

입소 시 분류 심사는 출소 시 **가석방 심사**까지 이어지는 데이터 사슬을 형성한다:

```
신입 분류심사 (KORI-PI + 고위험 여부 + RM 여부 결정)
    ↓ 처우 프로그램 배정 (고위험군 = 고강도, 저위험군 = 저강도; RNR Risk 원칙)
    ↓ 형의 일정 시점 도달 → 수급 분류 재심사 (KORI-PI 재실시 + 교도관 관찰 평가서 + 고위험·RM 평가서)
    ↓ 가석방(=지역사회전환 프로그램) 심사
       — KORI-PI, RM, 가족생활, 접견 빈도 등 모든 누적 데이터 활용
```

→ 분류는 단순한 경비처우 결정이 아니라 **교정 전 과정의 의사결정을 구조화하는 데이터 인프라**.

## 미래 쟁점과 과제 (PDF 슬라이드 11)

PDF 슬라이드 11이 제시한 4가지 과제:

1. **타당성 검증 강화**: 객관적 지표는 고정될 수 없다 — 시대·교도소·법령 변화에 맞춰 지속적 검증 필요
2. **총체적 분류 필요**: 보안 + 처우 + 재활을 **개별 평가가 아니라 통합적·총체적 접근**으로 융합 (현행은 개별 평가 중심)
3. **성별 문제**: 여성 재소자 등 특수집단에 적합한 분류체계 개발 (남성 중심 개발의 한계 명시)
4. **중간제재(intermediate sanctions) 활용 확대**: 단순 시설 분류를 넘어 **재활 중심 분류로의 전환**

### 추가 파생 쟁점 (강의 + 추측)

- **AI·알고리즘 기반 분류** (추측): 예측 정확도 향상 vs. 알고리즘 편향 위험
- **동적 재분류 주기**: 얼마나 자주 재평가해야 최적인가
- **과밀수용과 분류의 충돌**: 분류 결과와 달리 시설 과밀로 인해 적정 시설 배치 불가능

## 의외의 연결점

- **과밀수용 관리 ↔ [[팬데믹과-범죄]]**: COVID-19 기간 재판 지연 → 구치소 과밀화 심화 → 분류 체계 압박 증가. 팬데믹이 교정 기회구조에도 영향을 미쳤다.
- **분류 환경 영향 맹점 ↔ [[피해자-비난론]]의 체제 비난론**: 범죄자가 교도소 환경의 영향을 받듯, 피해자도 사회구조적 불평등의 영향을 받는다 — 개인 귀인보다 구조·환경 귀인이 중요하다는 같은 인식론적 관점.
- **객관성 추구 ↔ [[재범위험성평가]]의 세대적 한계**: 도구가 객관적이어도 사용되는 사회적 맥락이 편향을 만든다 (여성 수형자, 소수민족 과대표집 문제).
- **세계대전·대공황 → 객관적 분류제도** (추측): 20세기 초 노동력 부족이 재소자를 "무료 인력"으로 활용하게 만들었고, 이 노동 활용의 효율 극대화 요구가 분류 도구화의 1차 동인. 인권이 아니라 **경제적 효율성**이 객관적 분류제도의 산모. [[교도작업]]의 계약노동제·관용제 변천사와 한 묶음.
- **자원 부재 → 객관적 분류제도**: 객관적 분류제도는 "이상이 진보해서"가 아니라 "교정직 인력이 부족해서" 등장. 도구화는 자원 부족의 행정공학적 해답이다 — [[보호관찰]] 머신러닝 도구의 도입 논리(인력 부족 → 도구 의존)와 동형.
- **점수 척도의 방향 함정**: "점수가 높을수록 위험성이 낮음"인 척도 존재 — [[타당도와-신뢰도]]·[[척도]] 페이지가 다루는 점수 해석의 일관성 문제와 직결. 척도 운용자가 매뉴얼 없이 직관으로 해석하면 정반대 결과 도출 가능.
- **PCL-R = 구조화된 임상 판단의 전형**: [[사이코패시]] 진단도구가 동적 요인을 객관 측정하기 위해 시나리오를 설계한 사례 — 1세대 임상판단에서 2세대 보험계리적 판단으로 가는 중간 다리.

## 관련 개념

- [[재범위험성평가]] — 분류도구의 이론적 기반(RNR), 세대별 진화 (1세대 비구조화 임상판단 → 2세대 정적요인 보험계리 → 3세대 동적요인 통합)
- [[범죄자-진단평가]] — 분류심사의 상위 우산 개념. 정적/동적/복합 도구 분류
- [[교정처우론]] — 분류 결과에 따른 처우 프로그램 배정 (RNR Risk 원칙: 고위험 = 고강도, 저위험 = 저강도)
- [[교도소화]] — 교도소 환경이 분류 결과를 변화시키는 메커니즘
- [[교도작업]] — 20세기 초 객관적 분류제도 등장의 경제적 배경(노동력 활용)
- [[보호관찰]] — 출소 후 분류 정보 연계 활용
- [[젠더와-범죄]] — 여성 수형자 분류도구 부재 문제
- [[사이코패시]] — PCL-R = 구조화된 임상 판단의 전형
- [[타당도와-신뢰도]] — 분류도구의 타당성 검증, 척도 방향성 문제
- [[척도]] — 분류 점수 해석의 일관성 문제
- [[소년범-교정과-소년사법]] — J-DRAI(청소년 재범위험성 평가척도)는 객관적 분류제도의 데이터마이닝·지역 캘리브레이션 논리를 소년 보호관찰에 적용한 5세대 평가
