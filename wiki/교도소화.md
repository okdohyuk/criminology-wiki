# 교도소화 (Prisonization)

> **학습 takeaway**: 교도소화는 "신입 재소자가 교도소 문화를 학습하고 내면화하는 과정"이다. 일반 사회의 사회화와 달리 **거부권이 없는 강제적 상황**이므로, 순수한 "적응"이 아니라 **가치관의 범죄적·반사회적 변질**을 초래한다. Clemmer의 보편적 박탈 모델에서 출발해 Wheeler의 U자형 곡선과 Garabedian의 역할 분화로 발전했다는 점이 핵심이다.

## 정의

**Clemmer(1940)의 정의**:  
신입 재소자가 교도소라는 특수한 사회의 문화, 관습, 규범을 학습하고 내면화하는 과정. 어린이의 "사회화(socialization)"와 유사하나, **거부권이 없는 강제적 상황**이라는 점에서 근본적으로 다르다.

**핵심 특성**:
- 단순한 제도에의 적응 ✗
- 재소자의 가치관이 **범죄적·반사회적 방향으로 변화** ✓
- 교도소 문화의 부정적 영향력에 주목

## 이론 계보

### 1. Clemmer의 보편적 모델 (Universal Model)

**가설**: "수형 기간이 길수록 교도소화는 직선적으로 강화된다"

**보편적 요소** (거의 모든 재소자가 겪는 공통 변화):

| 요소 | 내용 |
|------|------|
| **비공식적 구금 지식 습득** | 교도소 운영의 실제 규칙, 권력 구조, 생존 전략 학습 |
| **열등적 역할 수용** | "수형자"라는 신분을 내면화. 정체성의 일부가 됨 |
| **새로운 사회적 습관 및 생존 기술** | 교도소 특화 언어(은어), 신체 언어, 관계 형성 기술 |

**한계**:
1. 수형 단계별 변화 간과 (입소/중기/출소의 차이)
2. 재소자의 **역할 차이** 무시 (주인공 vs. 약자)
3. 재소자의 **생활 지향성에 따른 동화 속도** 차이 무시

---

### 2. Wheeler의 U자형 곡선 (U-Shaped Curve Model)

**주장**: 교도소화는 직선이 아니라 **곡선**을 따른다.

```
친사회성
     높음  │         ╱╲          (출소 말기)
          │        ╱  ╲
          │       ╱    ╲
          │      ╱      ╲
   중간   │     ╱        ╲
          │    ╱          ╲
          │   ╱            ╲
     낮음  │__╱              ╲
         입소초기  수용중기   출소말기
```

**3단계 변화**:

| 단계 | 태도 | 설명 |
|------|------|------|
| **입소 초기** | 친사회적 | 아직 사회의 규범·가치관 유지. 사회와의 연결성 강함 |
| **수용 중기** | 반사회적 (정점) | 교도소 문화에 깊이 몰입. 교도소 규범 내면화. 가장 교도소화된 시점 |
| **출소 말기** | 친사회적 (회귀) | 다시 사회 규범을 지향. 출소 준비로 사회적 정체성 회복 |

**해석**: 이 곡선은 재소자가 "점진적으로 변질되지 않는다"는 의미 — 상황과 심리 상태에 따라 가변적이다.

---

### 3. Garabedian의 역할 분화 모델 (Role Differentiation Model)

**주장**: "모든 재소자가 U자형 곡선을 따르지는 않는다. 역할에 따라 교도소화 패턴이 다르다."

**재소자의 주요 역할 유형**:

| 역할 유형 | 교도소화 패턴 | 행동 특성 |
|---------|-----------|---------|
| **"정치인(Politician)"** | U자형이 아님 (수평) | 형기 내내 교도관의 규범에 순응. 권력 추구 없음. 낮은 교도소화 |
| **"주인공(Heavyweight/Convict)"** | 교도소화 높음 | 수형자 사회의 권력층. 집단 규범 선도. 높은 교도소화 |
| **"중간층"** | U자형 추종 | Wheeler의 곡선을 보임 |
| **"약자(Right guy)"** | 가변적 | 강자의 보호 필요. 역할 불안정 |

**핵심**: "같은 기간 수감되어도 역할에 따라 교도소화 정도가 완전히 다르다."

## 교도소화의 메커니즘

### 1. 박탈 모델 (Deprivation Model)

**Sykes의 5대 박탈**:

1. **자유(Liberty)의 박탈** — 신체 자유, 행동 제약
2. **자율성(Autonomy)의 박탈** — 모든 결정을 타자가 내림
3. **이성 관계(Heterosexual Relations)의 박탈** — 친밀감 결핍
4. **안전(Safety)의 박탈** — 폭력·학대 위험
5. **재화와 용역(Goods and Services)의 박탈** — 물질적 결핍

**심리적 반응**:
- 개별 대응 불가 → **집합적 대응**으로 전환
- 교도소 부문화 형성 (비공식 규범·계층·경제)

**"거부의 거부" 전략**:
- "나를 거부하는 사회를 내가 먼저 거부한다"
- 교도소 규범 수용으로 **자존감 회복** 시도

### 2. 유입 모델 (Importation Model)

**Irwin & Cressey의 주장**:  
"교도소 부문화는 교도소 안에서 자생한 것이 아니라, 재소자가 **사회에서 가져온 것**이다."

**재소자의 배경**:
- 사회적 낙오자, 범죄 경험, 하위계층 출신
- 교도소 입소 전부터 반사회적 가치관·행동 습관
- 교도소는 이러한 **기존 특성을 강화**하는 환경일 뿐

**함의**: 교도소화 = 과거 경험의 연장선, 단순한 적응 이상의 심화

### 3. 통합 모델 (Integration Model)

**주장**: "교도소 환경이 틀(frame)을 제공하고, 입소 전 개인적 특성이 내용(content)을 채운다."

- 박탈 모델과 유입 모델의 대립이 아니라 **수준이 다른 설명** — 매크로(환경) vs 마이크로(개인)
- 시설 환경(박탈의 강도·구조)이 비공식 조직의 전반적 형태를 결정
- 개인의 사전 사회화가 그 안에서 구체적 역할과 적응 방식을 결정
- **비총지화(Non-totalization)**: 비공식 권력이 단일 우두머리 중심으로 완전히 통합되지 않고 소규모 집단 권력으로 분산·병존

## 교도소화의 결과

### 개인 수준

| 결과 | 설명 |
|------|------|
| **정체성 변화** | "나는 범죄자"에서 "나는 수형자"로의 정체성 전환 |
| **규범 내재화** | 교도소 규범이 자신의 가치 체계가 됨 |
| **사회 복귀 어려움** | 출소 후 사회 규범 적응 불가 |
| **재범 위험** | 교도소 부문화에 깊이 빠진 자일수록 재범률 높음 |

### 조직 수준

| 결과 | 설명 |
|------|------|
| **비공식 조직의 강화** | 공식 규칙보다 비공식 규범이 실제 통제력 행사 |
| **수형자 계층의 고착** | 주인공/약자 구분이 영구화 |
| **교도관 권위 약화** | 비공식 규범이 교도관 지시를 초월 |
| **시설 통제 곤란** | 폭력·마약·성적 학대 증가 |

### 수용자 권력 구조의 역사적 변화

교도소의 비공식 통제는 실질적 수용자 리더에 의해 유지된다.

| 시대 | 리더 유형 | 권력 기반 | 통제 방식 |
|------|---------|---------|---------|
| **과거 왕형** | 폭력적 우두머리 | 신체적 힘·공포 | 다른 수용자를 힘으로 억압 |
| **현대 정치형** | 자원 배분자 | 담배·정보·편의품 통제 | 자원 재분배로 충성과 지지 획득 |

**질서 구매 메커니즘** (Guard-Inmate Bargaining):
```
소수 교도관 → 다수 수용자 직접 통제 불가
        ↓
우두머리 수용자에게 특권(눈감기) 제공
        ↓
우두머리가 수용자 집단 질서 유지
        ↓
교도관이 "질서를 구매"하는 구조
```

> 이 구조는 부패(corruption)와 달리 제도적 불가피성으로 이해된다 — 완벽한 규칙 집행은 소수 교도관으로 불가능하기 때문.

## 재소자 유형 분류

### Sykes의 재소자 부문화 유형

재소자 집단은 교도소 생활 지향성에 따라 세 가지 부문화(subculture)로 분화된다:

| 유형 | 지향 | 특징 |
|------|------|------|
| **범죄지향형** | 범죄 세계 | 교도소를 범죄 훈련소로 활용. 범죄 인맥 확장. 교도소화 정도 가장 높음 |
| **출형지향형** | 조기 석방 | 가석방에 집착. 교도관 비위 맞추기. 순응적 겉모습 |
| **합법지향형** | 사회 복귀 | 프로그램 참여·기술 습득·재통합 준비. 교도소화 정도 낮음 |

### Sykes의 재소자 역할 유형

교도소 내 비공식 역할은 집단 규범·자원 배분·생존 전략에 따라 분화된다:

| 역할 | 특징 |
|------|------|
| **고릴라(Gorilla)** | 힘으로 다른 재소자를 약탈·공갈. 가장 위험한 폭력적 유형 |
| **늑대(Wolf)** | 다른 재소자를 성적으로 강제 지배. 권력 표시 수단으로 성 사용 |
| **생짓(Rat)** | 교도관에게 다른 재소자 정보를 제공. 집단 내 가장 경멸받는 역할 |
| **상인(Merchant)** | 담배·먹을 것·정보 등 물자를 거래. 비공식 경제의 핵심 |
| **중심인 / 진정한 남자(Real Man)** | 강인함·자존심·비굴하지 않음으로 존경받는 이상적 수용자상 |
| **어리석은 파괴자(Ball Buster)** | 이유 없는 규칙 위반·교도관 도발. 집단에 무익한 갈등 유발 |
| **떠거리(Tough)** | 실제 힘 없이 허세로 위협. 집단에서 신뢰받지 못함 |

### Shemaleger의 현대 재소자 분류

현대 교도소 환경에 맞게 업데이트된 7가지 유형:

| 유형 | 특징 |
|------|------|
| **깔끔한 신사(Square John)** | 범죄와 거리를 두고 진짜 사회 복귀를 준비. 비범죄적 정체성 유지 |
| **기회주의자(Opportunist)** | 교도소 제공 기회(교육·훈련)를 자신에게 유리하게 활용 |
| **현실주의자(Realist)** | 교도소를 "어쩔 수 없는 현실"로 받아들이고 최소 마찰로 기간 채우기 |
| **쾌락주의자(Hedonist)** | 단기 쾌락(마약·도박·성)을 추구. 장기 목표 없음 |
| **식민자(Colonizer)** | 교도소를 편안한 '집'으로 인식. 출소를 원하지 않는 경우도 있음 |
| **종교인(Religious)** | 종교적 신앙을 통해 구원·개인 변화를 추구 |
| **변호인(Legalist)** | 법적 수단을 최대한 활용(이의 신청·소송·권리 주장). 제도 안에서 투쟁 |

### 남녀 교도소 문화 비교

| 구분 | 남성 교도소 | 여성 교도소 |
|------|-----------|-----------|
| **권력 구조** | 서열·힘의 논리 (강자가 약자를 지배) | 단계적 정서적 유대 중심 |
| **부문화** | 범죄지향·지배·약탈 중심 | 관계·가족적 유대 중심 |
| **교도관 관계** | 적대적·상호 독립 | 정서적 교류·의존 관계 빈번 |
| **특징적 집단** | 고릴라/상인/생짓 역할 분화 | 유사 가족 관계(pseudo-family) 형성 |

> 여성 교도소의 **유사 가족(pseudo-family)**: 재소자들이 어머니·딸·자매 등 가족 역할을 서로 모방하는 비공식 집단을 형성. 박탈 모델 중 "이성 관계의 박탈"에 대한 여성 특유의 집단적 대응.

## 의외의 연결점

- **[[낙인이론]]과의 차이**: 낙인이론은 사회의 반응으로 2차 일탈 발생을 설명하지만, 교도소화는 조직 내 **자발적 내면화** 강조
- **[[일상활동이론]]의 교도소 사례**: 동기화된 가해자(교도소 강자) + 적법한 표적(약자) + 보호자 부재(폐쇄적 환경) = 교도소 내 폭력·성적 학대 기회 구조
- **[[사회해체이론]]의 역설**: 교도소라는 공식 조직이 존재하는데도 비공식 자치 조직이 발달 → 공식성만으로는 질서 유지 불가
- **[[교도소화|우리 범죄인들의 세계]] ↔ [[재범위험성평가]]**: 높은 교도소화를 보인 재소자일수록 RNR 원칙의 "반응성(Responsivity)" 평가에서 저위험으로 잘못 분류될 위험
- **[[교정처우론]]과의 긴장**: 교화 프로그램(교정처우)은 개인 변화를 추구하지만, 교도소화는 집단 규범 강화로 개인 변화를 방해
- **질서 구매 ↔ [[사회계약론]]**: 교도관이 우두머리에게 특권을 주고 질서를 구매하는 구조는 홉스의 사회계약과 유사하다 — 강자가 보호를 제공하고 약자는 복종한다. 그런데 여기서 "강자"는 국가(교도관)가 아닌 수용자 우두머리다
- **현대 정치형 리더 ↔ [[사회자본]]**: 정보·자원 재분배로 충성을 이끄는 현대 우두머리는 사회자본 이론의 "자원동원 능력"과 동일한 구조 — 교도소 내부에 자원 네트워크가 형성됨
- **Wheeler U자형 곡선 ↔ [[가정폭력]]의 폭력 순환이론**: 둘 다 "시간에 따른 비선형적 변화 패턴"을 그린다. 중기에 반사회성이 정점에 달했다가 출소전 회귀하는 패턴은, 가정폭력에서 긴장→폭발→사과→긴장이 반복되는 순환과 구조적으로 유사 — 특정 시점의 스냅샷으로 전체를 판단하면 오해가 생긴다는 공통 교훈
- **유입 모델 ↔ [[가정폭력]] 경찰 개입 RCT**: 직장 없는 가해자에게 강력 대응이 효과 없는 이유를 유입 모델로 설명할 수 있다 — "잃을 것(직장·사회적 자본)이 없는 사람은 처벌 위협에 무반응." 교도소화와 가정폭력 개입 모두 사회적 자원 수준이 중재 변수임을 보여줌
- **남녀 교도소 문화 차이 ↔ [[가정폭력]] 피해자화 대물림**: 여성 재소자의 유사 가족 형성은 다수가 IPV 피해자임을 반영 — 피해자화 이력이 시설 내 관계 형성 방식에도 각인됨. "약한" 관계 형성 역량이 아니라 생존 전략으로 읽어야 한다는 페미니스트 교정학의 관점
