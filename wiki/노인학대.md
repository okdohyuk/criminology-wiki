# 노인학대 (Elder Abuse)

> **학습 takeaway**: 노인학대는 아동학대·가정폭력과 달리 **유교적 孝 사상**이 신고를 억제하는 추가적 은폐 메커니즘을 갖는다 — "자녀를 보호하려는 부모의 심리"가 피해 노인을 침묵하게 만든다. 중화이론(Tomita)이 노인학대에 적용되는 방식 — 가해자가 "나는 가난해서 어쩔 수 없었다"(책임 부인), "별거 아니다"(손상 부인), "노인이 도움 요청이 아니라 관심 끌기를 했다"(피해자 부인) — 는 범죄자의 자기합리화를 분석하는 가장 정교한 틀이다.

## 정의

노인에 대한 신체적·정신적·성적 폭력, 경제적 착취, 가혹행위, 유기 또는 방임.
**(노인복지법 제1조의2 제4호)**

- **노인**: Fattah & Sacco(1989) 분류 — 초기노령(64~74세), 고령(75세 이상), 초고령(85세 이상)
- **국제 비교**: 개념 자체가 사회문화적으로 다양 — 노르웨이(가족갈등 포함), 홍콩(노인유기 포함), 인도(며느리의 불경 포함)

## 유형 (중앙노인보호기관 분류)

| 유형 | 정의 | 주요 징후 |
|------|------|----------|
| **신체적 학대** | 물리적 힘·도구로 신체 손상 유발 | 설명 안 되는 상처, 묶인 흔적, 영양부족 |
| **정서적 학대** | 비난·모욕·위협으로 정서적 고통 유발 | 무기력, 눈물, 표정 없음, 극단적 행동 |
| **성적 학대** | 의사에 반한 강제적 성적 행위 | 걸을 때 어려움, 성병, 속옷 찢어짐 |
| **경제적 학대** | 의사에 반한 재산·권리 착취 | 재산 타인 명의 전환, 개인 귀중품 없어짐 |
| **방임** | 의식주·의료 의무 거부·불이행 | 영양실조, 오물 냄새, 의료 처치 없음 |
| **자기방임** | 스스로 최소한의 자기보호 포기 | 식사 거부, 치료 거부, 의도적 죽음 추구 |
| **유기** | 의존적 노인을 버리는 행위 | 낯선 장소 배회, 주소 모름 |

**학대 특성**: 지속성(5년 이상 34.3%), 복합성, 반복성, **은폐성** (자녀 보호 심리로 피해 노출 기피)

## 실태

- 공식통계: 여성노인이 남성노인보다 **2.5배** 이상 학대피해 높음
- 실태조사(설문): 남성 8.6% > 여성 6.2% — 공식통계와 역전
- 가해자와 피해자 관계: **직접 혈연(자녀)** > 며느리·사위
- 연령이 높아질수록 학대가 증가하지는 않으나, 85세 이상 방임 **20%** 내외
- 신고 건수: 12,009건 → 신고율 2.4% 미만 (엄청난 암수범죄)

**낮은 신고율 원인**:
1. 확인 가능한 증거 부족
2. 표준화된 검진도구 없음
3. 노인차별 및 신고기피
4. **유교(孝) 사상** — 자식 피해 불원, 수치심
5. 법노인병학 전문가 부족

## 이론적 설명

### 초기이론 (피해자·가해자 중심)

| 이론 | 핵심 주장 | 한계 |
|------|----------|------|
| **개인 간 관계론** | 노인·가해자 모두가 원인 | 일방적 귀인 위험 |
| **의존관계론** | 보호자에 대한 노인의 의존이 학대 위험 유발 | 신체적 학대 설명 불충분 |
| **보호자 스트레스론** | 부양 스트레스를 해소 못하면 공격적 반응 | 학대의 부분적 이유에 불과하다는 비판 |
| **폭력의 대물림론** | 피학대 아동이 성인 후 노부모를 학대 | 실증연구 드물다는 비판 |

### 범죄학 이론의 응용

| 이론 | 노인학대 적용 |
|------|------------|
| **억제이론** | 처벌 가능성·수준 낮음 → 학대 지속 |
| **긴장이론** | 보호자 스트레스 설명에 유용 (Payne, 2006) |
| **자아통제이론** | 자아통제력 낮은 부양자가 학대 |
| **사회학습이론** | 폭력의 대물림 메커니즘 설명 |
| **일상활동이론** | 요양원 학대: 동기화된 요양보호사 + 취약한 노인 + 감시 부재 |
| **중화이론** | 가해자의 자기합리화 5가지 기술 |

### 중화이론의 노인학대 적용 (Tomita)

| 중화기법 | 노인학대에서의 적용 |
|--------|-----------------|
| **책임 부인** | "가난해서, 부모 때문에 어쩔 수 없었다" |
| **손상 부인** | "별거 아니다, 심각한 것 아니다" |
| **피해자 부인** | "노인이 관심 끌기를 했을 뿐" |
| **비난자에 대한 비난** | "남의 가정사에 참견한다" |
| **더 높은 가치에 호소** | "배우자를 만족시키기 위해 해야 했다" |

## 발생 요인 4분류

| 요인 | 내용 |
|------|------|
| **피해노인** | 여성, 고령, 신체·정신건강 불량, 고립, 알코올 중독 |
| **부양자** | 정신적 문제, 냉담한 성격, 아동기 학대경험, 약물남용 |
| **가족상황** | 경제적 부담, 내키지 않는 부양, 부부 갈등 |
| **사회** | 노인차별주의, 핵가족화, 孝 의식 변화, 노인복지시설 부족 |

## 대책

- **노인학대 예방의 날**: 매년 **6월 15일** (노인복지법)
- **중앙노인보호전문기관** 설치·운영
- **신고의무자**: 의료인, 노인복지시설 종사자, 119구급대원, 가정폭력 상담소 종사자 등
- **지역노인보호전문기관 업무**: 신고전화 운영, 현장조사, 피해노인·학대자 상담

## 의외의 연결점

- **중화이론의 노인학대 적용 ↔ 화이트칼라범죄**: Sykes & Matza가 청소년 비행을 위해 개발한 중화이론이 노인학대 가해자 합리화에도 동일하게 작동한다 — 이론의 범용성 → [[화이트칼라범죄]]
- **유교 孝 사상 ↔ 범죄에-대한-두려움**: 孝 사상 때문에 신고하지 않는 노인 피해자는 "침묵하는 피해자"(미야자와 분류) — 사회문화적 규범이 범죄 은폐를 강화한다 → [[피해자-유형분류]]
- **요양원 일상활동이론 ↔ CPTED**: 요양원 내 학대 예방 = 공간 설계(CCTV, 다인실 배치)로 "감시의 존재"를 강화하는 것 → [[CPTED]]
- **보호자 스트레스 ↔ 일반긴장이론**: 부양 스트레스라는 긴장이 학대라는 비합법적 적응으로 이어지는 경로 — 애그뉴의 GST 3가지 긴장원천 중 "원하는 것을 얻지 못하는 긴장" → [[일반긴장이론]]
