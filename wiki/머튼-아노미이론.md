# 머튼의 아노미이론 (Merton's Anomie Theory)

> **학습 takeaway**: 머튼의 천재성은 **"범죄는 개인 결함이 아니라 사회가 만들어낸 압력의 결과"**라는 통찰을 **"문화적 목표 vs 제도적 수단의 괴리"**로 공식화한 데 있다. 5가지 적응 양식(순응/혁신/의례/도피/반역)은 외우기 쉬운 **정책 분석 도구**다 — 지금도 범죄율 국제비교에서 "불평등 구조 + 성공 이데올로기 과잉" 조합을 보면 이 이론이 작동한다. 핵심 질문: 한국 사회는 목표-수단 괴리가 얼마나 큰가? → 아노미가 진행 중이다.

## 이론 구조

### 거시: 아노미
- 사회는 **문화적 목표**(무엇을 원해야 하는가)와 **제도적 수단**(어떻게 얻어야 하는가)을 균형 있게 제공해야 함
- 목표만 지나치게 강조되고 수단은 계급별로 제한되면 → **목표-수단 괴리** → 아노미
- 아노미 = **규범의 규제력이 약화된 사회 상태**

### 미시: 긴장
- 개인은 이 구조 아래서 **긴장(strain)** 을 겪음
- 긴장 상황에 처한 개인은 5가지 적응 양식 중 하나를 택함

## 5가지 적응 양식 (외우기 필수)

| 양식 | 문화적 목표 | 제도적 수단 | 해당 예시 |
|------|------------|------------|----------|
| **순응(conformity)** | 수용 | 수용 | 대다수 시민 |
| **혁신(innovation)** | 수용 | 거부 → 불법적 수단 | **범죄** (핵심!) |
| **의례(ritualism)** | 거부 | 수용 | 목표 포기하고 규칙만 따름 |
| **도피(retreatism)** | 거부 | 거부 | 정신이상자·약물중독자·부랑자 |
| **반역(rebellion)** | 둘 다 거부 + 새로운 목표·수단 설정 | | 급진 운동가 |

- **범죄학적으로 중요한 것은 '혁신'과 '도피'**
- 혁신 = 범죄 / 도피 = 일탈

## 왜 하층이 가장 취약한가

- 문화는 **모든 계급에게 경제적 성공을 열망**하라고 압력
- 하지만 **제도적 수단(좋은 교육·직업)에 대한 접근은 계급별로 차별화**
- → 하층일수록 "혁신"(불법 수단)으로 기울어짐

## 미국 사회 비판 (1938)

- **열린 계급이동**이라는 이데올로기 = 환상
- 실제로는 심한 불평등 구조
- 이 **이데올로기 × 구조적 제약**의 결합이 범죄 문제의 근원
- → "경제적 성공을 지나치게 강조하는 문화 + 심한 불평등 = 범죄율 높음"

## 비판과 평가

### 허쉬의 경험적 반박
- Hirschi(허쉬의 사회유대이론)가 청소년 설문조사로 긴장이론 검증
- 결과: **교육적 열망이 높은 청소년일수록 오히려 비행을 덜 저지름**
- 오히려 **열망·기대 모두 낮은 청소년**이 비행 더 많음
- → 미시 수준에서 긴장이론은 지지되지 않음

### 반격
- 머튼 이론은 **원래 거시 수준**의 이론 (전체 사회 범죄율)
- 개인 적응 양식은 부차적
- **청소년의 사소한 비행이 아닌 전체 범죄율 비교가 본래 검증 대상**

### 적용 범위 확장
- "하층 범죄만 설명한다" → 상층도 **금전적 성공의 상대성** 때문에 혁신 동기 가능 → [[화이트칼라범죄]]에도 적용
- "재산범죄만 설명한다" → **좌절 → 파괴성**이라는 심리 메커니즘 포함하면 폭력 범죄도 설명 가능

## 의외의 연결점

- **한국의 부동산·입시 경쟁**: 보편주의(누구나 성공 가능)와 구조적 제약(계급 격차)의 병존 = 전형적 머튼식 아노미
- **[[화이트칼라범죄]] 설명**: 상층이 더 혁신 동기가 클 수 있음 — 끝없는 성공 열망이 이미 성취한 사람도 더 가지려 하게 만듦
- **[[약물-혐오-스토킹-환경범죄|약물]]과 도피**: 약물 중독자는 머튼의 '도피' 적응의 전형
- **[[조직범죄]]**: 혁신 양식의 집단적 제도화 — 불법 수단이 안정화된 구조
- **[[사회해체이론]]과 상보**: 같은 시대 같은 도시 문제를 다른 축(구조 불일치 vs 공동체 붕괴)으로 설명
