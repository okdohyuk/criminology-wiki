# 살인 (Homicide)

> **학습 takeaway**: 범죄학의 **기준점** 같은 범죄. 거의 모든 사회에서 가장 중대한 범죄로 다뤄지기 때문에 **국제비교 지표**로 자주 활용된다. 살인을 **형법상 유형 / 현장 특성 / 동기 / 피해자 수**의 4축으로 분류할 수 있다는 것만 잡으면 시험·실무 모두에서 정돈된 분석이 가능하다.

## 정의

타인의 생명을 고의로 단절하는 범죄. 과실치사·정당화 가능한 살해와는 구별된다. [[범죄의-층위|자연적 의미의 범죄]]의 대표적 예시.

## 1. 형법상 유형

| 유형 | 내용 |
|------|------|
| 살인죄 | 사람을 살해 — 기본 유형 |
| 존속살해 | 자기 또는 배우자의 직계존속을 살해 |
| 촉탁·승낙에 의한 살인 | 당사자 동의 있는 살인 |
| 자살교사·방조 | 타인을 교사 또는 방조하여 자살하게 함 |
| 위계·위력에 의한 살인 | 기만·강압으로 자살을 유도 |

> **⚠️ 2023년 8월 형법 개정**: **영아살해죄·영아유기죄 폐지** — 영아살해(10년 이하 징역)·영아유기(2년 이하)를 별도 감경하던 조항이 삭제되고, 일반 살인(5년 이상)·유기(3년 이하)로 처벌. "치욕 은폐를 위한 살인은 참작 사유"라는 구조가 70년 만에 폐지. 영아도 성인과 동등한 피해자로 보호.

### 영미법 vs 국내법 비교

| 영미 보통법 | 내용 | 국내 형법 대응 |
|------------|------|---------------|
| 1급 모살(1st-degree murder) | 살인 고의 + **사전 계획·준비** | 계획적 살인 → 중범죄결합·극단살인 유형 |
| 2급 모살(2nd-degree murder) | 살인 고의 있으나 사전계획 없음 | 보통동기살인 |
| 자발적 고살(voluntary manslaughter) | 고의 있으나 정상참작 | 참작동기살인 / 상해의도였으나 결과적 사망 |
| 비자발적 고살(involuntary manslaughter) | 살인 의도 없고 부주의로 사망 | **과실치사죄** |

국가는 생명침해를 세분화하여 다루되, 결국 **생명**이라는 법익을 핵심 보호대상으로 삼는다.

## 2. 범죄자·현장 특성

| 구분 | 특징 |
|------|------|
| **조직적 살인** | 청부살인형, 계획성 높음, 증거인멸 치밀 |
| **비조직적 살인** | 개인적 원한·분노·좌절, 우발적, 면식관계 다수 |

현장이 말끔하면 조직적, 혼란스러우면 비조직적일 가능성이 높다.

## 3. 동기 유형

- **도구적 살인** — 금전·성적 욕구·다른 범죄 완수의 수단
- **표출적 살인** — 분노의 폭발
- **이상동기 살인** — 동기 불명확, 불특정 다수 대상 → **범죄두려움**을 크게 자극

## 4. 피해자 수

- **일반살인** — 1명
- **대량살인** — 한 장소에서 다수 살해
- **연속살인** — 짧은 간격 내 이동하며 살해
- **연쇄살인** — 사건 사이에 **심리적 냉각기** 존재 (핵심 구분점)

**연쇄살인범 하위유형**: 망상형 / 사명형 / 쾌락형 / 권력형

## 양형기준상 분류 (양형위원회)

| 유형 | 명칭 | 권고형 범위 |
|------|------|------------|
| 제1유형 | 참작동기살인 | 4~6년 |
| 제2유형 | 보통동기살인 | 10~16년 |
| 제3유형 | 비난동기살인 | 15~18년 |
| 제4유형 | 중대범죄결합살인 | 20~25년 |
| 제5유형 | 극단적 인명경시살인 | 사형·무기·25년 이상 |

- **제5유형**은 불특정 다수를 향한 무차별 살인(2명 이상 살해)이 해당 → [[이상동기범죄]]와 직접 연결
- 권고형 외에 감경·가중 구간이 별도 존재하므로 실제 선고형은 범위 바깥이 될 수 있음

살인은 **결과**만이 아니라 **동기와 맥락**에 따라 법적·사회적 의미가 달라진다.
