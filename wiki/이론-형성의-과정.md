# 이론 형성의 과정 (Theory Building)

> **학습 takeaway**: 범죄이론은 **초자연적 설명**(악령·저주)에서 **자연적·경험적 설명**으로 넘어오면서 정교화됐다. **가설→명제→이론**의 3단계와 **이론 형성 5단계**를 한 번에 외우면 논문 읽는 눈이 바뀐다. 좋은 이론의 4조건까지 기억하면 완성.

## 사회과학 이론의 특성: 개연성·확률·경향성

> **핵심 구분**: 사회과학은 자연과학의 **필연 법칙**("물은 100℃에서 반드시 끓는다")이 아니라 **개연성·확률·경향성**의 설명을 한다.

"어떤 조건에서 어떤 결과가 나타날 **가능성이 높아지는가**"를 밝히는 것이 목표.
- ❌ 필연법칙: "A이면 반드시 B이다" — 사회과학에서는 거의 불가능
- ✅ 개연성 명제: "A가 높을수록 B가 높아질 경향이 있다" — 검증 가능하고 반례도 존재

이 특성 때문에 범죄이론은 "왜 누군가는 범죄를 하고 다른 사람은 안 하는가"를 설명하지, "이 사람이 반드시 범죄를 한다"를 예측하지 않는다.

## 이론이란 무엇인가

> **핵심 정의**: 이론 = **개념들 간의 관계를 구조적으로 설명하는 체계** = **변수들 간의 관계를 연결하는 틀**

- 단순한 설명문이 아니라, 변수 A → 변수 B → 변수 C 를 인과적으로 연결하는 구조
- **예**: 일반긴장이론 = "긴장(부정적 사건) → 분노(부정적 감정) → 비행"의 3단계 인과 경로
- **예**: 일상활동이론 = "동기화된 범죄자 + 적절한 표적 + 보호자 부재 = 범죄 발생"

> ⚠️ **흔한 오류**: 논문 제목은 "비행친구와 사이버불링"인데, 이론은 "일상활동이론(피해선택이론)"을 쓰는 경우. 이론이 내 변수들을 연결하는 틀이 되어야 하는데, 전혀 무관한 이론을 칸 채우기용으로 쓴 것. 이론을 정확히 이해하면 이런 불일치가 눈에 바로 보인다.

## 개념 3단계

| 수준 | 의미 |
|------|------|
| **가설(Hypothesis)** | 두 변수 관계에 대한 검증 가능한 진술 |
| **명제(Proposition)** | 반복 검증으로 지지된 진술 |
| **이론(Theory)** | 여러 명제가 연결된 설명 체계 |

이론은 **단일 문장이 아니라, 관련 명제들이 하나의 설명 구조**를 이루는 상태.

## 이론 형성 5단계

1. **가설 설정** — 기존 이론을 바탕으로
2. **자료 수집** — 검증을 위한 데이터
3. **자료 분석** — 변수 간 관계 확인
4. **가설 검증** — 지지 여부 판단
5. **이론 확장·수정·폐기** — 결과에 따라

이 과정은 학술 절차만이 아니라 **범죄정책의 방향을 바꾸는 실질적 과정**.

## 좋은 이론의 4조건

| 조건 | 의미 |
|------|------|
| **논리적 일관성** | 내부 모순 없음 |
| **경험적 검증 가능성** | 자료로 참·거짓 확인 가능 |
| **경험적 타당성** | 실제 자료에 의해 뒷받침 |
| **정책적 함의** | 범죄를 줄이는 실천적 단서 제공 |

결국 좋은 범죄이론은 **설명으로 끝나지 않고, 범죄를 줄이기 위한 실천적 단서까지 제공**해야 한다.

## 이론의 독창성 — 무엇이 새로운 이론을 만드는가

> **핵심**: 많은 범죄학이론은 내용이 겹친다. **새로운 핵심 명제(proposition)**를 제시할 때 비로소 독창적인 이론으로 인정받는다.

- ❌ 기존 개념의 재조합·강조만 바꿈 → 독창성 인정 어려움
- ✅ 기존 이론이 설명하지 못한 관계를 명시적 명제로 제시 → 독창적 이론

**예시: [[제도적-아노미이론]]**
- 머튼의 아노미이론(경제적 긴장)을 계승하지만
- "**경제제도가 다른 사회제도(가족·교육·정치)를 지배할 때 범죄율이 높아진다**"는 명제를 새로 추가
- 이 **"제도적 불균형"** 명제가 독창성의 근거

→ 실천적 함의: 범죄이론을 읽을 때 "이 이론이 추가한 핵심 명제가 무엇인가"를 먼저 파악하라.
