# 이론 형성의 과정 (Theory Building)

> **학습 takeaway**: 범죄이론은 **초자연적 설명**(악령·저주)에서 **자연적·경험적 설명**으로 넘어오면서 정교화됐다. **가설→명제→이론**의 3단계와 **이론 형성 5단계**를 한 번에 외우면 논문 읽는 눈이 바뀐다. 좋은 이론의 4조건까지 기억하면 완성.

## 사회과학 이론의 특성: 개연성·확률·경향성

> **핵심 구분**: 사회과학은 자연과학의 **필연 법칙**("물은 100℃에서 반드시 끓는다")이 아니라 **개연성·확률·경향성**의 설명을 한다.

"어떤 조건에서 어떤 결과가 나타날 **가능성이 높아지는가**"를 밝히는 것이 목표.
- ❌ 필연법칙: "A이면 반드시 B이다" — 사회과학에서는 거의 불가능
- ✅ 개연성 명제: "A가 높을수록 B가 높아질 경향이 있다" — 검증 가능하고 반례도 존재

이 특성 때문에 범죄이론은 "왜 누군가는 범죄를 하고 다른 사람은 안 하는가"를 설명하지, "이 사람이 반드시 범죄를 한다"를 예측하지 않는다.

## 이론이란 무엇인가

> **핵심 정의**: 이론 = **개념들 간의 관계를 구조적으로 설명하는 체계** = **변수들 간의 관계를 연결하는 틀**

- 단순한 설명문이 아니라, 변수 A → 변수 B → 변수 C 를 인과적으로 연결하는 구조
- **예**: 일반긴장이론 = "긴장(부정적 사건) → 분노(부정적 감정) → 비행"의 3단계 인과 경로
- **예**: 일상활동이론 = "동기화된 범죄자 + 적절한 표적 + 보호자 부재 = 범죄 발생"

> ⚠️ **흔한 오류**: 논문 제목은 "비행친구와 사이버불링"인데, 이론은 "일상활동이론(피해선택이론)"을 쓰는 경우. 이론이 내 변수들을 연결하는 틀이 되어야 하는데, 전혀 무관한 이론을 칸 채우기용으로 쓴 것. 이론을 정확히 이해하면 이런 불일치가 눈에 바로 보인다.

## 개념 3단계

| 수준 | 의미 |
|------|------|
| **가설(Hypothesis)** | 두 변수 관계에 대한 검증 가능한 진술 |
| **명제(Proposition)** | 반복 검증으로 지지된 진술 |
| **이론(Theory)** | 여러 명제가 연결된 설명 체계 |

이론은 **단일 문장이 아니라, 관련 명제들이 하나의 설명 구조**를 이루는 상태.

## 이론 형성 5단계

1. **가설 설정** — 기존 이론을 바탕으로
2. **자료 수집** — 검증을 위한 데이터
3. **자료 분석** — 변수 간 관계 확인
4. **가설 검증** — 지지 여부 판단
5. **이론 확장·수정·폐기** — 결과에 따라

이 과정은 학술 절차만이 아니라 **범죄정책의 방향을 바꾸는 실질적 과정**.

## 좋은 이론의 4조건

| 조건 | 의미 |
|------|------|
| **논리적 일관성** | 내부 모순 없음 |
| **경험적 검증 가능성** | 자료로 참·거짓 확인 가능 |
| **경험적 타당성** | 실제 자료에 의해 뒷받침 |
| **정책적 함의** | 범죄를 줄이는 실천적 단서 제공 |

결국 좋은 범죄이론은 **설명으로 끝나지 않고, 범죄를 줄이기 위한 실천적 단서까지 제공**해야 한다.

## 이론의 독창성 — 무엇이 새로운 이론을 만드는가

> **핵심**: 많은 범죄학이론은 내용이 겹친다. **새로운 핵심 명제(proposition)**를 제시할 때 비로소 독창적인 이론으로 인정받는다.

- ❌ 기존 개념의 재조합·강조만 바꿈 → 독창성 인정 어려움
- ✅ 기존 이론이 설명하지 못한 관계를 명시적 명제로 제시 → 독창적 이론

**예시: [[제도적-아노미이론]]**
- 머튼의 아노미이론(경제적 긴장)을 계승하지만
- "**경제제도가 다른 사회제도(가족·교육·정치)를 지배할 때 범죄율이 높아진다**"는 명제를 새로 추가
- 이 **"제도적 불균형"** 명제가 독창성의 근거

→ 실천적 함의: 범죄이론을 읽을 때 "이 이론이 추가한 핵심 명제가 무엇인가"를 먼저 파악하라.

## 사례: [[고전주의-범죄학]] vs 초자연적 설명

- **초자연**: 악령·사탄·마녀 → 검증 불가
- **고전주의**: 합리적 계산 → 검증 가능
- [[합리적-선택-이론]]: 변수 통제된 검증 가능

## 범죄학 이론과 사회의 불가분성 — 3원칙

PDF 강의 핵심: 범죄학 이론은 **진공 상태에서 만들어지지 않는다**. 사회적 맥락이 이론의 형태를 결정한다.

### 원칙 1. 개인의 경험이 이론에 투영된다
- 이론 개발자의 출신·성향·정치관·가치관이 어떤 현상에 주목할지를 결정
- 시골 출신 학자가 도시에 와서 본 일탈 → 그것이 "비정상"으로 인식 → 이론으로 발전
- 도시에서 나고 자란 사람에게는 같은 현상이 "자연스러움" → 인식되지 않음

### 원칙 2. 사회 변화가 이론적 패러다임을 바꾼다

| 시기 | 사회 맥락 | 범죄에 대한 설명 방식 |
|------|---------|----------------|
| **식민지·중세** | 종교가 가치규범의 중심 | 범죄 = 악마의 충동, 범죄유혹의 결과 |
| **1800년대 후반** | Darwin 진화론·사회적 다윈주의 | 범죄자 = 덜 진화된 자, 격세유전·열등 자질 (Lombroso) |
| **1960년대 미국** | 호황기, 아메리칸드림 | "왜 노력해도 부자가 못 되어 범죄로 가는가" → [[아노미-긴장이론]] |
| **2000년대 이후** | 정치적 보수주의로 회귀(Reagan·Bush) | 응보·처벌 강조 → [[신고전주의-범죄학]] |

### 원칙 3. 절대 진리는 없다 — 상대적 설득력만 있다

- 범죄학 인사이클로피디아(백과사전) 기준 약 **300개 이론** 존재
- 학부 수준은 10~15개, 핵심 6대 이론
- 새 이론이 계속 나오는 이유: 범죄 현상을 단 하나의 관점에서 완전히 설명하는 이론은 없음
- 각 이론은 위·아래·옆에서 본 부분적 진실
- → "검증 안 되는 이론을 폐기시키는 작업"이 학계의 정상적 과제

## 형사정책으로의 연결 — 원인 진단이 대책을 결정한다

같은 범죄 현상도 **원인 진단**이 다르면 처방이 정반대로 나온다 (PDF 슬라이드):

| 원인 진단 | 대표 이론 | 대책 |
|---------|---------|------|
| 유전적 비정상(교정 불가) | [[생물학적-범죄이론]] 강한 형태 | 구금·격리·극단적 통제(과거 불임수술까지) |
| 정신질환 | [[심리학적-범죄이론]] | 정신치료·치료적 개입 |
| 경제적 박탈에 의한 스트레스 | [[일반긴장이론]]·[[아노미-긴장이론]] | 직업훈련·교육·취업기회 제공 |
| 사회 무질서·규범 해체 | [[사회해체이론]] | 거시적 지역사회 강화(체육시설·청소년단체) |
| 비행친구·환경 학습 | [[차별접촉이론]]·[[사회학습이론]] | 환경 분리·긍정적 모델 노출 |

→ **이론 선택은 결국 정책 선택**이다. "어떤 이론을 받아들이는가"는 가치판단의 영역.

### 20세기 형사정책의 진자운동

1. **20세기 초**: 범죄자 = 야만적 진화단계로 회귀한 격세유전적 퇴행자·정신박약자 → 불임수술도 불사하는 **극단적 통제**
2. **20여 년 후(중기)**: 범죄자 개인보다 **병리적 생활환경 문제**가 대두 → 빈곤층 청소년에 대한 **구원** 중요
3. **최근(1970s 이후)**: 지나친 관용주의가 오히려 범죄의 원인 → **엄격한 처벌 강조, 교도소 수용**이 답

→ 같은 사회가 100년 안에 정반대 정책을 오간다는 사실 자체가, 범죄학이론이 **사회적 산물**임을 보여준다.

## 신주류범죄학과 중범위적 해결책

최근 트렌드: 거시적 사회개조보다 **중범위(meso-level) 접근** 선호.
- 거시적 접근(전체 사회 변혁) → 비용 크고 효과 불확실
- 중범위적 접근 — 특정 공간·특정 집단·소규모 지역사회에서 실효성 있는 프로그램
- 예: 범죄 다발 시간·장소에 경찰 배치(거점순찰), 좁은 표적 개입

**대증요법 vs 근본요법 논쟁**:
- 대증요법(증상 대응): 약 처방, 경찰 배치 → 효과 즉각적이나 근본 미해결
- 근본요법(원인 제거): 식단 교정, 빈곤 해소 → 효과 장기적, 비용·시간 큼
- **현대 범죄학은 대증요법을 선호** — 범죄가 이동(displacement)하더라도 0이 되는 곳이 생기면 사회 전체 이익이 0보다 크다는 논리
- 비판적 관점: 범죄가 **약한 동네로 옮겨가는 불평등** 문제 잔존
