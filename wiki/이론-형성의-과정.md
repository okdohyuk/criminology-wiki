# 이론 형성의 과정 (Theory Building)

> **학습 takeaway**: 범죄이론은 **초자연적 설명**(악령·저주)에서 **자연적·경험적 설명**으로 넘어오면서 정교화됐다. **가설→명제→이론**의 3단계와 **이론 형성 5단계**를 한 번에 외우면 논문 읽는 눈이 바뀐다. 좋은 이론의 4조건까지 기억하면 완성.

## 개념 3단계

| 수준 | 의미 |
|------|------|
| **가설(Hypothesis)** | 두 변수 관계에 대한 검증 가능한 진술 |
| **명제(Proposition)** | 반복 검증으로 지지된 진술 |
| **이론(Theory)** | 여러 명제가 연결된 설명 체계 |

이론은 **단일 문장이 아니라, 관련 명제들이 하나의 설명 구조**를 이루는 상태.

## 이론 형성 5단계

1. **가설 설정** — 기존 이론을 바탕으로
2. **자료 수집** — 검증을 위한 데이터
3. **자료 분석** — 변수 간 관계 확인
4. **가설 검증** — 지지 여부 판단
5. **이론 확장·수정·폐기** — 결과에 따라

이 과정은 학술 절차만이 아니라 **범죄정책의 방향을 바꾸는 실질적 과정**.

## 좋은 이론의 4조건

| 조건 | 의미 |
|------|------|
| **논리적 일관성** | 내부 모순 없음 |
| **경험적 검증 가능성** | 자료로 참·거짓 확인 가능 |
| **경험적 타당성** | 실제 자료에 의해 뒷받침 |
| **정책적 함의** | 범죄를 줄이는 실천적 단서 제공 |

결국 좋은 범죄이론은 **설명으로 끝나지 않고, 범죄를 줄이기 위한 실천적 단서까지 제공**해야 한다.
