# 재범위험성평가 (Recidivism Risk Assessment)

> **학습 takeaway**: 재범위험성평가는 "교정이 효과 없다(Martinson, 1974)"는 선언에 대한 반격으로 탄생했다. Gendreau & Ross(1979, 1987)의 "Something works!"가 RNR 원칙으로 이어지고, 이 원칙이 평가도구 세대를 진화시켰다. **1세대(임상판단) → 2세대(정적 요인) → 3세대(정적+동적) → 4세대(평가+처우계획 통합) → 5세대(머신러닝)**라는 계보가 핵심이다. 한국의 JDRAI-S(2세대), KPRAI-R(3세대)가 이 흐름에 위치한다. 국외 사례 6종(SIR·OASys·LS/CMI·HCR-20·VRAG·PCL-R + 성범죄 4종: RRASOR·Static-99/2002·SORAG)을 통해 일반/폭력/성폭력 도구의 분기와 정적·동적·임상 요인 결합 방식이 어떻게 진화했는지 파악할 수 있다. 핵심 통찰: **PCL-R 요인2(사회적 일탈)가 요인1(대인관계·정서)보다 일반재범 예측력이 압도적**(.37** vs .02) — 사이코패시의 "정서적 무공감"보다 "행동패턴"이 재범의 실질 신호.

## 핵심 개념

- **위험성(Risk)**: 개인이 미래에 다른 범죄를 저지를 가능성
- **위험(Dangerousness)**: 피해자에게 닥친 위험하거나 불행한 결과 포함 (Clark, 1999)
- **재범위험성 평가**: 앞으로 일어날 해로운 행동의 개연성·빈도·영향 확률을 계산하는 것 (Kemshall, 1996)

## 역사적 배경

| 시기 | 사건 |
|------|------|
| 1880~1960 | 미국 사회복귀의 시대 — 교정처우 프로그램 확장 |
| **1928** | **Burgess**: 최초 보험계리적 가석방 예측도구 개발 (21개 변인) |
| 1960s | **Vera Institute**: 보석 대상자 평가도구 — 동적 요인 최초 포함 |
| 1972 | 미연방 가석방 심사위원회 **특이요인점수(Salient Factor Score, SFS)** 개발 |
| 1970s | **위스콘신 WRA**(Wisconsin Risk Assessment): 보호관찰 재범위험성 평가도구 |
| **1974** | **Martinson: "Nothing works!"** — 교정처우의 비효과성 비판 |
| 1980~1990 | 형사정책 강경 보수화 |
| 1980s | **LSI(Level of Service Inventory)**: RNR 기반 3세대 도구의 원형 (LSI-R의 전신) |
| **1979, 1987** | **Gendreau & Ross: "Something works!"** — 효과적 프로그램 존재 입증 |
| 1990 | **RNR 원칙** 체계화 (Andrews, Bonta & Hoge) |
| 2000s | **COMPAS**: 4세대 도구, 100개+ 문항, 사회학습이론 기반 |
| 2010s | 머신러닝 기반 5세대 도구 출현 |

## RNR 원칙 (Risk-Need-Responsivity)

재범방지에 효과적인 처우의 3원칙 (Andrews, Bonta & Hoge, 1990):

| 원칙 | 핵심 질문 | 내용 |
|------|---------|------|
| **R (Risk)** | Whom to target? | 고위험군에 집중 개입; 저위험군 고강도 프로그램은 역효과 |
| **N (Need)** | What to change? | 범죄유발욕구(criminogenic needs)를 처우 목표로 삼아야 함 |
| **R (Responsivity)** | How to do it? | 범죄자 학습능력에 맞는 처우 — 일반: CBT / 특정: 개별 장벽 제거 |

### 위험성 원칙 (Risk Principle) 세부 기준
- 고위험군 선별 → 상응한 집중 개입
- 구체 기준: 기간 **6~9개월 + 대상자 시간의 40~70% 점유**
- **저위험군에게 고강도 프로그램 = 재범률 증가 역효과** (주의)

### Central 8 — 욕구 원칙의 핵심 요인

| 분류 | 요인 | 설명 |
|------|------|------|
| **Big 4** (범죄의 직접 원인) | 반사회적 태도 | 범죄를 정당화하는 인지 왜곡 |
| | 반사회적 성격 | 충동성·공격성·반사회적 성격장애 |
| | 반사회적 친구 | 범죄 친화적 대인관계망 |
| | 비행경력 | 과거 범죄·비행의 이력 |
| **Next 4** (미시 환경) | 가족 갈등 | 가족 관계 불안정 |
| | 불안정 직장/학교 | 교육·고용 실패 경험 |
| | 약물남용 | 알코올·마약 의존 |
| | 여가생활 | 반사회적 여가(비행 친구와의 시간) |
| **비범죄유발요인** (개입 효과 낮음) | 자아존중감 | 개입해도 재범 감소 효과 미미 |
| | 불안·우울 | 정신건강 문제 (직접 연관 논쟁) |
| | 사회계층 | 빈곤 자체는 재범 예측력 낮음 |
| | 성공 동기부여 | 단독 개입으로는 부족 |

→ **비범죄유발요인에만 집중하는 처우는 효과 없음** — Big 4·Next 4가 핵심 타겟

### 반응성 원칙 (Responsivity Principle) 세부

| 유형 | 내용 | 예시 |
|------|------|------|
| **일반 반응성** | 인지행동치료(CBT) 활용 | 모든 범죄자에게 기본 적용 |
| **특정 반응성** | 대상자별 처우 장벽 제거 | 낮은 지능·정신질환·문맹·문화적 장벽 개별 대응 |

### 한국의 교정 분류 도구: REPI

- **REPI**(교정재범예측지표): 신입심사·재심사에서 재수용 위험등급 판정
- 정기재심사 시점: 형기의 1/3, 1/2, 2/3, 5/6 도달 시
- 평가 결과 활용: 처우 계획 수립 → 가석방 신청 → 보호관찰 단계까지 연계

#### 위험관리 등급 (Dangerous Offender Classification)

강력범죄(성폭력·방화·강도·살인) 사건으로 분류센터 정밀 분류를 받은 재소자에게 부여:

```
강력범죄 입소 → 분류센터 정밀심사 (PCL 포함)
        ↓
위험관리 등급 부여 → 수감 기간 내내 지속
        ↓
정기 재심사 때마다 위험관리 재평가
        ↓
"안전하다" 판정 → 가석방 심사 대상 진입 가능
```

- REPI(일반 분류) + 위험관리 등급(강력범죄 특별 추적)의 이중 체계

#### PCL(사이코패시 체크리스트)의 가석방 전 활용

가석방 **직전**에 PCL 점수를 추가 확인하는 이유:
- REPI로 위험성 점수가 낮아졌더라도, 사이코패시적 성격이 남아 있으면 재범 위험
- PCL이 묻는 것: "이 사람이 가석방 후 자유로운 환경에서도 반사회적 성격·무공감성을 유지하는가?"
- 분류센터에서 범죄 유형별 특화 도구(성범죄 → Static-99R, 반사회성 → PCL-SV)를 함께 활용

## 재범위험성 평가도구의 세대별 발전

### 1세대 — 비구조화된 임상적 판단

- 임상전문가의 주관적 판단 (MMPI 등 심리평가도구 병행)
- **구조화된 임상판단(Structured Clinical Judgement, SCJ)**: 시나리오 기반 결정 트리로 신뢰도 향상 시도 → 여전히 주관적 해석 여지 있음
  - 작동 방식: "대상자 답변이 A 정도면 → B 질문으로 이동, C 정도면 → D 질문" 형태의 의사결정 트리. 동등한 훈련을 받은 임상가라면 누구든 비슷한 [[타당도와-신뢰도|타당도·신뢰도]]를 유지하도록 분기를 미리 고정 — 1세대를 "최대한 표준화"하려는 시도
  - 강의자 정의: **타당도** = "검사가 측정하려는 것을 실제로 측정하는가" (재범위험성 척도가 재범을 제대로 평가하는가) / **신뢰도** = "누가 채점해도 같은 답이 나오는가"
- **단점**: 예측타당도 낮음, 기저율 정보 부족, 평가자마다 다른 결과, 법정 제출 객관적 증거 부족 (Monnahan, 1981)

### 2세대 — 정적 위험요인 기반 보험계리적 판단

- 성별, 연령, 범죄경력, 과거 수감경험 등 **변화하기 어려운** 개인 요인
- 최초: 1930년대 Burgess(1928) 가석방 예측도구 (21가지 변인)
- 1972년 미연방 가석방 심사위원회 "특이요인점수(salient factor score)" 개발
- **보험계리적 판단**: 보험 계약 시 가족력이 있으면 암 발생 확률이 높다고 판단하는 방식과 동일 — 통계 기반 확률 계산
- **장점**: 예측타당도 높음, 채점 쉬움, 평가자 간 신뢰도 높음, 서류만으로 평가 가능, 담당자 훈련 부담 적음
- **치명적 단점**: **동적 요인 없음 → 교정처우의 대상·목표·강도 결정 불가**
  - 위험성만 알 수 있을 뿐 "뭘 개입해줄지", "반사의적 성향을 개선해줄 건가, 대인관계 기술을 훈련시켜줄 건가" 전혀 알 수 없음
  - 결국 개별맞춤 처우 수립 불가 → 3세대의 필요성 발생
- **한국**: JDRAI-S

### 3세대 — 정적 + 동적 위험요인

- **범죄유발욕구(criminogenic needs)** = 변화가능한 역동적 위험요인 추가
  - 반사의적 성향, 분노 조절 능력, 피해자 공감 능력, 친구관계, 가족관계, 자존감, 자아상 등 → 프로그램으로 변화 가능

**동적 위험요인의 세부 구분 (Harris & Hanson, 2010)**:

| 유형 | 정의 | 예시 | 처우 함의 |
|------|------|------|---------|
| **급성요인** (Acute Factors) | 빠르게 변화하는 단기 위험 상태 | 약물사용, 분노, 피해자 접근성 | 즉각 개입 신호 |
| **안정요인** (Stable Factors) | 비교적 지속적이나 처우로 변화 가능 | 성에서의 자기관리, 사회적 영향, 애착 | 장기 처우 목표 |

- 대표적 도구: **SONAR**(Sex Offender Needs Assessment Rating) — 급성·안정 동적요인 체계 적용

**보호요인(Protective Factors) 논쟁**:
- 보호요인 = 재범위험성을 감소시킬 가능성을 가진 요인
- 핵심 쟁점: "위험요인과 명확히 분리되는 보호요인이 존재하는가?" (Hoge, Vincent & Guy, 2012)
- 존재한다면 기존 위험요인 척도와 어떤 관계로 설정할지 방법론 논쟁 진행 중
- 일부 3~4세대 도구에서 보호요인 항목 포함 시도

- 주기적·재평가 가능 → 프로그램 효과 측정 가능
- **장점**
  - 프로그램 효과를 재평가 점수 변화로 측정 가능
  - 개별맞춤 처우 계획 수립 가능 (위험성이 높으면 고강도 프로그램, 기간 연장)
  - 처우와 재범 감소의 연결고리 파악 가능
- **단점 인과사슬** (PDF 6주차 명시): 동적 문항 = **추상적 구성개념**(반사회적 태도·분노조절 등) 측정 → **평가자 간 신뢰도(interrater reliability) ↓** → **결과적으로 예측타당도 ↓** — 추상성이 신뢰도를 거쳐 타당도까지 끌어내리는 단방향 사슬. 매뉴얼 정교화·평가자 훈련이 필수 이유.
- **대표 도구**: **LSI-R**(Level of Service Inventory-Revised), PCRA(미국), OASYS(영국)
- **한국**: KPRAI-R(Korean Probationers Risk Assessment Inventory)

### 4세대 — 평가 + 처우계획 통합

- 3세대 평가결과와 처우계획 수립의 연계성 강화
- **대표 도구**: 
  - **LS/CMI**(LSI-R + 사례관리, 캐나다) — Andrews & Bonta(RNR 창시자) 개발
  - **COMPAS**(100개 이상 문항, 사회학습이론 기반, 미국)
  - **OACS**(영국 법원) — 현존하는 평가도구 중 **가장 많은 동점 요인 12개**, 예측력 .67 (67%)
- COMPAS는 북미에서 LSI-R과 함께 가장 많이 사용되는 4세대 도구

> **한국의 4세대 공백**: 강의자 명시 — 한국은 **4세대 도구가 없다**. 2/3세대(REPI, JDRAI-S, KPRAI-R)에서 보호관찰의 5세대(머신러닝 전산시스템)로 점프. 평가-처우계획 통합 단계를 제도적으로 거치지 않은 채 알고리즘 기반 자동화로 직행 — 4세대의 핵심 가치(임상가가 평가 결과를 처우계획에 직접 반영하는 사례관리)가 한국 교정 시스템에 비어 있는 영역.

### 5세대 — 머신러닝 기반

- 대규모 데이터 + 머신러닝 기법 → 실무자 주관 개입 최소화
- 전통적 통계분석 한계 극복 → 재범예측력 향상
- 새로운 형태 2010년대 이후 출현
- **한국**: 보호관찰 단계에서 이미 도입 — 전산화 시스템으로 대상자·보호관찰관 양방 입력 후 자동 점수화

## 특수집단별 위험성 평가 도구 — 한국 현황 (2026년 기준)

일반 도구(REPI, KPRAI-R)는 성인 남성 범죄자 기준으로 개발. 특수집단에는 별도 도구 필요.

| 특수집단 | 한국 도구 현황 | 비고 |
|---------|------------|------|
| **청소년** | 있음 (JDRAI-S 청소년 버전) | 성인과 범죄 유발 욕구 구조 다름 |
| **정신질환 범죄자** | 있음 — 타당도 미검증 | 최근 개발, 효과성 연구 부족 |
| **성범죄자** | 있음 (Static-99R 등 활용) | 해외 도구 번역 적용 |
| **여성범죄자** | **없음** | 특화 척도 개발 전 단계 |
| **물질남용자** | 개발 중 | 미완성 |
| **디지털 성범죄자** | **없음** | 전통 성범죄자 도구로 대체 — 한계 인식 |
| **이상동기 범죄자** | 최근 개발 | 보호관찰 단계 활용, 검증 진행 중 |

> **맹점**: 5세대 도구가 존재해도 특수집단에는 적용 불가 — 훈련 데이터가 해당 집단을 반영하지 못하면 예측력 무의미. 여성범죄자·디지털 성범죄 도구 공백이 특히 시급한 과제.

#### 여성범죄자 전용 도구 — WRNA (미국)와 한국 공백

성중립적 도구(LSI-R 등)는 "전체 범죄자(=대부분 남성)"를 재범 고/저위험으로 가르는 요인을 추출하므로, 표본의 절대다수가 남성이라 **여성 고유 특성이 도구에 반영되지 못한다**(Belisle, Keen, Trejbalová et al., 2022). 여성만 따로 분리해 고위험/저위험 집단을 비교하면 다른 욕구가 드러난다(Salisbury, Belisle, Mercier & Prince, 2023):

- **여성 범죄유발욕구**: 역기능적 친밀 관계·학대/트라우마/범죄피해·낮은 사회적 지지·낮은 인적 자원 (관계 기반) ↔ 남성은 반사회적 성격·반사회적 동료
- **WRNA(Women's Risk Need Assessment)** — Van Voorhis(2008, 신시내티대): 처음엔 성중립 도구의 **보조도구(Trailer version)** → 이후 **독립도구(Standalone version)**로도 개발(Van Voorhis et al., 2010). 아동학대·성인학대·부모 스트레스·자기효능감·가족지지 등 표준 도구에 없는 관계·외상·양육 변수 포함.
- **반응성(R)의 재해석**: 여성에게는 학습능력뿐 아니라 **양육 등으로 인한 프로그램 참여 가능성** 자체가 반응성(Boppre, 2019) → 보호관찰 단계보다 출석이 보장되는 교도소 단계 처우가 유리할 수 있음.
- **한국**: 여성 특화 도구 부재 → 상세 [[여성범죄자-교정프로그램]].

#### 중간 제재 (Intermediate Sanctions) — 재범위험성 기반 처우 배치

분류 결과가 단순히 경비등급만이 아니라 **제재 수준** 결정으로도 이어짐:

```
[교도소] ←─────────────────→ [사회 자유]
  최고 보안 → 일반 수용 → 중간처우의 집 → 개방교도소 → 가석방 → 보호관찰 → 완전 석방
```

**중간처우의 집 (Halfway House)**:
- 낮에는 외부 사업장 출근, 밤에는 시설 복귀
- 적용 대상: 가석방 시기는 이르지만 교도소 과밀 해소 필요 시
- 한국 현황: 제도는 존재하나 거의 활용되지 않음 (법감정·사회적 수용성 문제)
- 효과: 사회 복귀 연습 + 과밀수용 완화 + 경제 활동 가능

### 평가도구 구성의 메타 이슈

**가중치 방법론** (송중일 외, 2007)
- **개별 가중치(optimal weights)** vs **동일 가중치(equal weight)** — 어느 쪽이 우세한지 통일된 결론 없음
- 동일 가중치 채택 도구의 평정 방식 3가지:
  1. yes-no 이분형 (특정 문항 해당여부만)
  2. 0-1-2 세분화 (질적 평가 일부 항목)
  3. 0-1-2-3-4 리커트 (완전한 질적 평가)

**준거변수(Response variable)의 단계별 분화** (Hamilton et al., 2017)
- 보석 등 재판 전: **도주위험성·주거환경** 평가 추가
- 시설내 처우: **시설내 규칙 위반 가능성** 추가
- 사회내 처우(보호관찰): **준수사항 위반 가능성** 추가
→ 같은 "재범위험성"이라도 형사사법 단계마다 측정하는 행동이 다르다 = [[조작적-정의와-변수]]의 단계별 분화

**판별력(discrimination)의 측정 — ROC/AUC 세부**
- 2×2 분류표(TP/FP/TN/FN) → 민감도·특이도 → AUC
- AUC 의미: "재범자 집단에서 무작위 표집된 대상자가 비재범자 집단의 대상자에 비해 평가점수가 높을 확률"
- 강의자 직관 해석: **.50 = 랜덤(동전던지기), .67 = 랜덤보다 +17% (중간 수준), .71 이상 = 높음**
- AUC는 지역의 재범기저율과 독립적 → 관할 간 비교 가능 = 판별력의 표준화 메트릭으로 채택된 이유

**평가자 간 신뢰도(interrater reliability)** — ICC(Intra Class Correlation)
- 선행연구에서 상대적으로 덜 다뤄지는 지표
- 동일 도구라도 사용되는 관할/환경에 따라 신뢰도 상이 → 도구의 "보편적 신뢰도"란 없다

### 일반 재범 도구 (브로드밴드)

#### 1. SIR (Statistical Information on Recidivism, Nuffield 1982)
- **국가**: 캐나다 교정국
- **세대**: 2세대 (정적 요인만)
- **항목**: 15개 (본 범의 내용·초범 시 연령·과거 수용경력·가석방 유무·탈옥 전력·위험수 분류 경험·최초 기소 시 연령·폭행 전력·혼인상태·합산형 청기·과거 성폭행·과거 가택침입·체포 당시 고용상태 등)
- **점수**: -30 ~ +27, 5개 집단(Very Good·Good·Fair·Fair to Poor·Poor)
- **재범 조작화**: 출소 후 3년 내 기소 가능한 범죄로 재체포
- **예측력**: 캐나다 비원주민 남성 AUC = .745 / 원주민 남성 AUC = .683 (Nafekh & Moriuk, 2002)
- **개정판**: SIR-R1
- → 원주민 집단에서 예측력 하락 = **알고리즘의 집단별 편향** 사례, [[낙인이론]]과 알고리즘 형사사법의 인종 편향 논쟁의 캐나다판

#### 2. OASys (Offender Assessment System, 1999)
- **국가**: 영국 이민국(Home Office)
- **세대**: 3~4세대 (현존 도구 중 최다 동적 요인)
- **활용**: 교도소·보호관찰 모두 사용. **영국 법정은 모든 범죄자에게 OASys + PSR(판결전조사서, Pre-Sentence Report) 요구** → 판결 단계의 표준 도구
- **구조**: 12개 영역, 영역마다 다른 가중치(범죄 경중에 따라 차등)
- **사회적 역동요인** 포함 → 성범죄 재범 예측력 증가 (May, 1999)
- **예측력**: 살인·상해 AUC = .67 / 살인·폭력 AUC = .66 (Debidin, 2009)
- **점수체계**: 각 항목 0(문제없음)·1(약간)·2(심각)
- **구조화된 면담 매뉴얼**: 비공개 — 영역마다 핵심질문 + 후속 탐색질문 제공

#### 3. LS/CMI (Level of Service/Case Management Inventory)
- **개발**: Andrews, Bonta & Wormith (2007) — RNR 창시자 직접 개발
- **계보**: LSI-R(1995 개정판)의 후속 = **LSI 계열 가장 최신**
- **세대**: **4세대의 전형** — 평가 + 사례관리·치료계획·서비스 전달 질 향상까지 통합
- **국가**: 캐나다 (지역별 전산화 시스템으로 구현)

**11개 영역 구조**

| 영역 | 핵심 내용 |
|------|---------|
| 1. 일반적 위험/욕구 요소 | 범죄력, 교육/직업, 가족/결혼, 여가/오락·동료, 알코올/약물, 친범죄적 태도/관심, 반사회적 양상 |
| 2. 세부적 위험/욕구 요소 | 범죄를 야기하는 잠재력·개인적 문제 (심리·정신건강·자살자해·스트레스·정신지체·분노조절) |
| 3. 수감경험 — 제도적 요소 | 수용력, 석방 방해요소 |
| 4. 기타 피검자 요소 | 사회·건강·정신건강, 피해자화 문항 |
| 5. 특수 반응성 고려사항 | 개입 효과를 저해하는 부차적 문제 (Responsivity 연계) |
| 6. 위험/욕구 개요 및 철회 | 피검자 기반·임상적·행정적 철회, 최종 LS/CMI 수준 |
| 7. **위험/욕구 프로파일** | **8개 범죄유발욕구 영역별 점수** (범죄이력·교육고용·가족혼인·여가오락·동료·알코올약물·반사회적 태도·반사회적 패턴) — 점수 가장 높은 영역 = 개입 최우선 타겟 |
| 8. 프로그램/배치 결정 | 수용·지역사회 감독·판결전 조사서 결과 요약 |
| 9. 사례관리 계획 | 행정 개요, 프로그램 목표·중재 계획 |
| 10. 경과 기록 | (재평가 추적) |
| 11. 석방 개요 | 석방/완료 유형 |

**4세대의 캐나다적 구현 — 강의자 설명**
> "3세대 도구에서 동적 요인을 알았으면 그걸 기반으로 수업·프로그램에 배정하면 되는데, 캐나다는 아예 **검사 도구 자체에 사례관리 계획·프로그램 배치 결정·경과 기록까지 넣어놨다**. 컴퓨터에서 클릭 클릭하면 처우계획이 나온다. 지역에 맞게 맞춤화되어 있다."

→ **한국 보호관찰소도 지역별 위험성 구간이 다르다**: 법무부에서 표준 척도가 내려와도 대전·서울 등 보호관찰소별 위험성 구간이 조정됨. 지역 문화·인구 구성이 개입되기 때문 — **알고리즘 도구도 "지역 캘리브레이션"이 필수**라는 실무 통찰.

**장점**
1. 전산화된 채점 프로그램 → 사용 편리
2. 우수한 신뢰도·타당도 실증 검증
3. 모든 형사절차(교도소·가석방·보호관찰)에서 포괄 사용
4. 성별·목적(시설 수용 vs 지역사회 감독) 따른 규준 보유

### 폭력범죄 도구 (네로우밴드)

#### 1. HCR-20 (Historical-Clinical-Risk Management-20)
- **개발**: Webster, Eaves, Douglas & Wintrup (1995)
- **방법**: SPJ (구조화된 전문가 평가) — 보험계리적이 아닌 임상 + 구조 결합
- **20개 항목 = 정적 10 + 임상 5 + 위험관리 5**

| 정적(Historical) H1~H10 | 임상(Clinical) C1~C5 | 위험관리(Risk Management) R1~R5 |
|---|---|---|
| H1 과거 폭력행위 존재 | C1 통찰력 결핍 | R1 결핍 요소 실현 가능성 계획 |
| H2 첫 폭력범죄 당시 나이 | C2 부정적 태도 | R2 불안정한 요소 노출 |
| H3 불안정한 대인관계 | C3 주요 정신장애의 활동적 증상 | R3 개인적 지지자 결핍 |
| H4 고용 문제 | C4 충동성 | R4 치료 시도 거부 |
| H5 물질 관련 문제 | C5 치료 반응성 | R5 스트레스 |
| H6 주요 정신질병 | | |
| H7 정신병질(사이코패스) | | |
| H8 초기 부적응 | | |
| H9 성격 장애 | | |
| H10 이전 감독(보호관찰 등) 실패 | | |

- **변별 기준점**: 20점 이상 = 폭력위험성 증가 (Douglas, Cox & Webster, 1999)
- **C3 주요 정신장애 활동적 증상의 폭력 관련 예시**:
  - 망상(Delusions): 피해망상으로 자신을 방어하려 선제 공격
  - 환청(Auditory Hallucinations): 명령환청(Command Hallucinations)에 따름
  - 심한 와해된 사고/행동: 예측불가능 + 위험행동
  - 급성 조증 증상: 판단력 저하·과도한 흥분·과대사고
- **한국 적용**: 한국판 HCR-20 타당도 연구 (최금화, 2009) — 병역심사에서 폭력행위 가능성 예측 양호

#### 2. VRAG (Violence Risk Appraisal Guide)
- **개발**: Harris, Rice & Quinsey (1993)
- **표본**: 고위험군 치료감호소 남성 범죄자 618명, 평균 7년 추적 → 폭력 재범과 유의 관련 변인만 선택
- **방법**: 보험계리적 (이론·모형 기반 X) — **변수 선정이 데이터 기반**
- **VRAG의 폭력 조작적 정의**: 사람 대상 모든 범죄, 폭행·무기 강도·신체 접촉 성폭력·살인·살인미수 / **심한 협박·비접촉 성범죄는 제외** (Barbaree et al., 2001)
- **12개 항목, 총점 -26 ~ +38, 9개 위험성 범주**
- **첫 항목이 PCL-R 점수** (최대 +12점, 가장 높은 가중치) — 사이코패시가 VRAG의 단일 최강 예측변수
- 다른 핵심 항목: 초등학교 시절 부적응, DSM 성격장애, 강력범죄 연령, 16세까지 양친 생존, 과거 조건부 석방 위반, 비폭력 범죄 점수(합산), 결혼 경험, DSM 정신분열증, 피해자의 심각한 피해, 알코올 문제 과거력, 여성 피해자
- 성격장애 가중치 차등: **A군(편집·분열) / B군(반사회·경계·자기애·연극) / C군(회피·강박·의존)**, B군이 폭력 위험성 가장 높음

#### 3. PCL-R (Psychopathy Checklist-Revised)
- **개발**: Hare (1991)
- **방법**: 훈련받은 전문가가 반구조화 질문지 + 공적 기록 기반 면담·채점
- **20문항 0-1-2점 / 총점 40점**
- **2요인 4단면 구조**:

| 요인 | 단면 | 핵심 내용 |
|------|------|----------|
| **요인 1 (대인관계·정서)** | 단면 1: 대인관계 | 입심 좋음/피상적 매력, 과도한 자존감, 병적 거짓말, 남을 잘 속임/조종 |
| | 단면 2: 정서성 | 후회·죄책감 결여, 얕은 감정, 냉담/공감능력 결여, 자신 행동 책임감 없음 |
| **요인 2 (사회적 일탈)** | 단면 3: 생활양식 | 자극 욕구/쉽게 지루해 함, 기생적 생활방식, 현실적·장기적 목표 부재, 충동성, 무책임성 |
| | 단면 4: 반사회성 | 행동통제력 부족, 어릴 때 문제행동, 청소년 비행, 조건부 석방/유예 취소, 다양한 범죄력 |

- **개발 당시 목적**: 정신병질적 성격 평가
- **재범예측력 검증** (Salekin, Rogers & Sewell, 1996; Barbaree et al., 2001)
- **PCL-R은 일반재범 예측에 중간 수준 유용 / 요인2가 심각한 폭력·성적 재범 예측에 중간 수준 유용**
- Hare(1998, 1999) **타당도 검증된 다른 위험도구와 함께 사용 권장**
- **한국에서도 활용** — 분류센터 정밀심사·가석방 전 검토에 PCL-R/PCL-SV 적용

**평가도구별 재범예측력 비교** (PDF 강의자료)

| 평가도구 | 일반 재범 | 심각한(폭력·성적) 재범 | 성범죄 재범 |
|---------|---------|--------------------|-----------|
| VRAG | **.45**\*\* | .24\*\* | .11 |
| RRASOR | .14\* | .20\* | **.26**\*\* |
| Static-99 | .34\*\* | .28\*\* | .18\* |
| SORAG | **.47**\*\* | **.30**\*\* | .14\*\* |
| PCL-R 전체 | .30\*\* | .17\* | .09 |
| PCL-R 요인1 | .02 | -.02 | .10 |
| **PCL-R 요인2** | **.37**\*\* | **.22**\*\* | .06 |

→ **결정적 통찰**: 사이코패시 전체 점수(PCL-R)보다 **요인2(사회적 일탈)만 떼서 보면 일반 재범 예측력이 .37로 가장 높다** (전체 PCL-R .30, 요인1 .02). 즉 **재범의 진짜 신호는 "냉담·공감결여"가 아니라 "충동성·무책임·반사회적 행동패턴"**. 임상적 직관(반사회적 성격의 차가움)과 통계적 실재가 어긋나는 지점 — [[사이코패시]] 페이지의 핵심 함의.

### 성폭력범죄 도구

#### 1. RRASOR (Rapid Risk Assessment for Sexual Offence Recidivism)
- **개발**: Hanson (1997)
- **목적**: 성범죄 재범위험성 신속 평가 — **기록조사만으로 평가 가능**
- **4개 항목**: 범죄자 나이(24세 이하), 성범죄 전력, 피해자와의 관계, 남성 피해자 유무
- **채점**: 성범죄 전력만 0~3점, 나머지 3항목은 각 1점 / 총 0~6점
- **AUC**: .52 ~ .77 다양 — 표본 따라 변동 큼
- **비판**: 아동성추행범 점수가 일반 강간범보다 높게 나옴 → 도구가 "위험"의 의미를 어떻게 정의하는지 비판 직면

#### 2. Static-99
- **개발**: Hanson & Thornton (2000)
- **구성**: RRASOR + SACJ (Structured Anchored Clinical Judgement scale, Grubin 1998) 보충
- **10개 정적 위험요인** (나이·친밀한 파트너 동거 2년 이상 경험·강력 비성적 폭력 유죄·이전 비성적 폭력·이전 성범죄·이전 선고 개수·비접촉 성범죄 유죄·피해자가 가족 외부인·낯선 사람 피해자·남성 피해자)
- **총점 위험등급**: 0-1 매우낮음 / 2-3 낮음 / 4-5 높음 / 6 이상 매우높음
- **예측력**: 성범죄 재범 AUC=.73(Beech et al., 2000) / 스웨덴 성범죄 AUC=.76·비성적 폭력 .74 / 덴마크 .64·.67
- **한국**: 수형자 대상 활용

#### 3. Static-2002
- **개발**: Hanson & Thornton (2003)
- **개선점**: Static-99 재구성, 평가 준거 일관성 개선, 개념적 명료성·일치도 추가
- **17개 항목**: 출소 후 나이·지속적 성범죄력·이전 성범죄 선고·소년전과 성범죄 체포·성범죄 선고율(15년당)·일탈적 성기호·비접촉 성범죄·남성 피해자·어리며 가족 외 피해자·피해자 관계·낯선 피해자·일반 범죄력·모든 범죄 체포/선고경력·가석방 실패·석방 후 강력범죄 기간·과거 비성적 폭력범죄 유죄
- **예측력**: Static-99 대비 폭력범죄 재범 예측 AUC 통계적으로 유의하게 높음 (Hanson & Thornton, 2003)
- **치료과정 종료자·비참여자에 높은 예측력 / 중단자 예측 정확도 감소** (Boer 2003; Haag 2005) — **치료 중단 자체가 새로운 위험 신호**

#### 4. SORAG (Sex Offender Risk Appraisal Guide)
- **개발**: Quinsey, Harris, Rice & Cormier (1998)
- **배경**: VRAG의 낮은 성범죄 재범 예측력(.11)을 보완하기 위해 개발
- **14개 항목** — 그중 9개는 VRAG과 동일, 추가 5개:
  - 이전 성범죄 유죄판결 수 (소년·근친상간 포함)
  - 14세 이하 소녀 대상 성범죄 (가해자-피해자 5살 이상 차이 시 +4)
  - 일탈적 성 기호 — **PPG(Penile Plethysmography, 음경측정기) 검사 결과 포함**: 이상성향 없음(-1) / 적용불가(0) / 이상성향 발견(+1)
- **총점 -17 ~ +34, 위험도 3개(낮음·보통·높음)**
- **예측력 검증** (Bartosh et al., 2003; Nunes et al., 2002)
- → **PPG 포함**은 SORAG의 가장 논쟁적 요소. "신체 반응을 위험 변수화"하는 윤리적·법적 쟁점 발생 (알고리즘 편향 논쟁의 신체 측정판)

### 도구 선택의 실무적 함의

| 시점 | 권장 도구 조합 |
|------|-----------|
| 입소 시 (브로드밴드) | SIR / LS/CMI / OASys |
| 폭력 이력자 (네로우밴드) | + HCR-20 또는 VRAG |
| 성범죄자 (네로우밴드) | + Static-99/2002 또는 SORAG |
| 사이코패시 의심 | + PCL-R (요인2 가중) |
| 가석방 직전 재평가 | + PCL 추가 + 동적 요인 재측정 |

> **다중 도구 사용의 이유**: 단일 도구는 예측력 한계 (AUC .67~.75) — 브로드밴드 + 네로우밴드 + 임상 요인 결합으로 예측력 보강. Hare(1998)의 "PCL-R은 다른 위험도구와 함께 사용 권장" 원칙이 모든 4세대 운영의 표준이 됨.

## 국제 비교: 판결 전 조사(Pre-Sentence Investigation)와 위험성평가

**영미법 vs. 대륙법의 차이**

| 법계 | 판결 전 조사 | 위험성평가 | 판사의 권한 | 사례 |
|------|-----------|----------|----------|------|
| **영미법** (미국, 영국) | 필수 실시 | 조사서에 포함·기반 | 위험성평가 결과로 형량 결정 | 영국 OACS 기반 선고 |
| **대륙법** (한국, 일본) | 형식적 | 교정 단계에서 주로 실시 | 전통적 재량 (죄질, 전력, 연령, 합의 등) | 한국 JDRAI-S, KPRAI-R |

**영국의 사례**
- **OACS**(Offender Assessment System): 법원 필수 활용 도구
- 판사가 판결 전 조사서를 받으면, OACS 기반 재범위험성 평가를 고려해 형량 결정
- 동점 요인 12개 (인관관계, 주거환경, 생활양식, 사고와 행동, 물질남용, 정서문제, 태도, 피해자 인식 등) 포함

**한국의 현황**
- 판결 전 조사는 형식적 수준
- 위험성평가는 **교정 단계**(입소 시, 교도소 내 재심사, 가석방 심사)에서 실시
- 법원은 전통적 요소 중심으로 판결 (위험성평가가 판결에 직접 영향 미치지 않음)

## 한국 위험성평가 제도

**근거법**: 형의 집행 및 수형자의 처우에 관한 법률 제59조 (2007.12.21 제정, 2008.12.22 시행)

**역사적 배경**
- 1960~70년대 초: 교정기관의 주관적 분류 제도 비판
  - 수용자들이 실제 위험성보다 높게 측정됨
  - 보안 수준이 매우 높음 → 교도관 과부하
  - 개인의 주관적 판단·차별성 문제
  - 한편 과도하게 낮게 분류되면 조기 가석방 → 재범률 상승
- 1980년대 이후: 과학적 분류 제도의 필요성 대두 → 현재의 법제 형성

**현재 시스템**
- 모든 수형자: 입소 시 신입심사 + 수용 중 재심사
- 분류처우회의 → 개별 분류 및 처우 결정
- 고려 항목: 성장 과정, 학력·직업경력, 범죄 경력, 정신 상태, 자력 개선 의지, 석방 후 생활 계획

**분류에 따른 실제 처우 차이** (경비등급별)
- **면접·접견**: 등급에 따라 1일 1회, 월 5회, 월 정기 등으로 제한
- **외부통신**: 전화 허용 여부·횟수, 영상통화, 편지 등 차등 적용
- **외출·외부활동**: 작업반출, 사회 활동(종교, 상담 등) 허용 여부
- **물품 지급·소지**: 개인 물품 허용 범위 (사진, 라디오, 외부 영상물 등)
- **거실 운영**: 거실문 개방, 자유활동 시간, 통영문 허용 등
- **가석방 신청**: 위험등급이 낮을수록 가석방 심사 대상 → 실제 출소 시점 결정
- **교육·작업·상담 프로그램**: 위험등급이 높을수록 강도 높은 프로그램 배정

→ 결국 **분류 = 경비등급 결정 = 자유도 결정**이 교정 현장에서의 핵심

## 평가도구의 적용 범위: 브로드밴드 vs 네로우밴드

(Brennan, 2017)

| 구분 | 브로드밴드(Broadband) | 네로우밴드(Narrowband) |
|------|---------------------|----------------------|
| **대상** | 일반 재범 전체 | 특정 범죄 유형(성폭력·폭력) |
| **용도** | 교정처우 배치, 보호관찰 강도 결정 | 특정 위험 사범 집중 관리 |
| **도구 예** | LSI-R, COMPAS | Static-99R(성범죄), HCR-20(폭력) |
| **장점** | 범용성 높음 | 특정 범죄 예측력 높음 |
| **단점** | 특수 집단 예측력 저하 가능 | 다른 범죄 유형에 적용 불가 |

> 실무: 입소 시 LSI-R(브로드밴드)로 전체 재범위험 평가 후, 강력범죄 이력이 있으면 Static-99R 등 네로우밴드 도구 추가 사용

## 예측타당도 평가 기준

(Helmus & Babchishin, 2017)

| 기준 | 개념 | 측정 방법 |
|------|------|----------|
| **판별력(Discrimination)** | 고위험 집단과 저위험 집단을 얼마나 잘 구별하는가 | ROC-AUC (0.5=무작위, 1.0=완벽) |
| **교정력(Calibration)** | 예측 확률이 실제 재범률과 얼마나 일치하는가 | 예측 50%→실제 재범 50% 수준 맞아야 함 |

- AUC .67 이상이면 실용적 예측력 있음으로 판단 (OACS 등)
- 판별력이 높아도 교정력 낮으면 → 위험군 분류는 잘 하지만 실제 비율 과대/과소 추정 → 처우 강도 오배치 문제

## 준거변수 (Criterion Variables)

재범을 어떻게 측정하느냐에 따라 예측도구 성능 평가가 달라짐 (Hamilton et al., 2017):

| 준거변수 | 특징 | 편향 |
|--------|------|------|
| **재체포(Rearrest)** | 범위 넓음, 기소 전 단계 포함 | 경찰 활동 수준에 따라 과다/과소 측정 |
| **재유죄판결(Reconviction)** | 법원 공식 기록 | 형사절차 탈락(검찰 기소 유예 등) 누락 |
| **재수용(Reincarceration)** | 가장 엄격 | 위반 사유 확인 안 됨 (기술적 위반 포함 가능) |

- 재반론: Labrecque et al.(2014) — 재평가 시 동적 문항 점수 변화가 재범을 유의미하게 예측

### 강의자 정리: 두 관점의 양립

> "**위험성 측정만 두고 보자면** 정적 요인이 통계적으로 재범자/비재범자를 가르는 변수만 추린 것이라 보험계리적 확률 게임에서는 충분히 정확하다. 그러나 **교정 처우 수립의 측면에서는** 동적 요인이 없으면 '뭘 개입할지' 결정할 수 없다 — 그래서 3·4세대가 도입되었지만, '재범 예측 정확도 향상' 자체에 동적 요인이 결정적으로 기여하느냐는 여전히 합의가 없다."

→ 핵심 통찰: **"예측"과 "처우"는 다른 작업**. 2세대로 예측 가능, 3·4세대로 처우 설계 가능. 동적 요인은 후자를 위한 것이지 전자의 정확도를 반드시 높여주진 않는다 — 두 목적을 혼동하면 도구 평가 자체가 어긋난다. → [[조작적-정의와-변수]]에서 "측정하려는 개념을 무엇으로 정의하는가"의 문제

## 의외의 연결점

- **PCL 가석방 전 활용 ↔ [[사이코패시]]**: 가석방 직전 PCL을 추가 사용하는 이유 — REPI 점수가 낮아도 사이코패시 성격이 잔존하면 출소 후 반사회적 행동 지속. 사이코패시가 "재범예측의 블랙홀"로 불리는 이유
- **위험관리 등급 ↔ [[낙인이론]]**: 강력범죄자에게 수감 내내 붙어다니는 위험관리 등급은 제도적 낙인 — 이 낙인을 벗기 위해 재소자가 "안전하다"는 증명을 수차례 해야만 가석방 진입 가능
- **Martinson "Nothing works" ↔ 사회과학의 개연성 명제**: "아무것도 효과 없다"는 선언은 과학적 명제가 아닌 도덕적 선언에 가깝다 — 이론의 형성 과정에서 개연성과 필연성을 구분해야 하는 이유 → [[이론-형성의-과정]]
- **RNR 원칙 ↔ 보호관찰**: RNR에 근거한 처우계획이 보호관찰의 처우 모형을 결정 → [[보호관찰]]
- **머신러닝 5세대 ↔ 합리적-선택-이론의 위기**: 알고리즘이 "이 사람은 재범할 것"이라고 예측하면, 그 예측 자체가 처우를 결정하고 결과를 바꾼다 — 예측의 자기실현·자기무효화 문제
- **LSI-R 동적 요인 논쟁 ↔ 낙인이론**: 재범위험성 평가 결과 자체가 낙인으로 작용해 재범을 촉진할 수 있다 — 예측 도구가 낙인 생성 도구가 되는 역설 → [[낙인이론]]
- **재복역률 범수별 데이터 ↔ 억제이론 한계**: 5범 이상 재복역률 59.9% — 반복범죄자에게 형사처벌이 억제 효과를 내지 못한다는 실증적 증거. 억제이론이 가정하는 "처벌 경험 → 범죄 억제"가 반복수감자에게는 역전된다 → [[억제이론]]
- **피해자촉발이론 ↔ RNR 욕구 원칙**: 볼프강의 피해자촉발이론이 범죄 과정에서 상호작용을 강조하듯, RNR의 욕구(Need) 원칙도 범죄자의 내적 상호작용 요소(반사회적 태도·친구관계)를 정밀 타겟팅해야 처우가 효과적임을 강조 — 두 이론 모두 "범죄는 일방향이 아니다"는 인식 → [[피해자-가해자-상호작용이론]]
- **PCL-R 요인2 우위 ↔ [[사이코패시]] 임상상의 재구성**: "사이코패시는 차가운 정서(요인1)이 본질"이라는 임상 직관과 달리, 실제 재범 예측은 행동패턴(요인2: 충동성·무책임·반사회성)이 압도. 사이코패시의 "위험성" 정의를 정서적 결함이 아닌 **행동적 일관성**으로 재구성해야 할 통계적 근거 — Hare 자신도 PCL-R 단독 사용을 권장하지 않는 이유
- **SIR 원주민 AUC 하락(.745 → .683) ↔ 알고리즘 편향**: 캐나다 SIR이 원주민 집단에서 예측력 떨어짐 = 훈련 데이터가 다수집단(비원주민)에 편향. 한국에서도 여성·디지털 성범죄 도구 부재 + 5세대 머신러닝 도입은 동일한 집단별 편향 문제를 반복할 위험. COMPAS 인종 편향 논쟁(ProPublica 2016)의 캐나다·한국 적용 가능성
- **SIR 원주민 편향 ↔ [[여성범죄자-교정프로그램|WRNA 성중립 도구 편향]] (동일 구조)**: WRNA가 필요한 근본 이유 — "성중립 도구는 남성 다수 표본 추출 요인이라 여성 고유 특성 미반영" — 은 SIR 원주민 AUC 하락과 *완전히 같은 메커니즘*이다. 다수집단(남성/비원주민) 데이터로 만든 도구를 소수집단(여성/원주민)에 적용하면 판별력이 떨어진다. 한쪽은 *인종* 축, 다른 쪽은 *젠더* 축의 같은 훈련데이터 편향. 그래서 Van Voorhis는 여성만 따로 분리해 고/저위험을 비교하는 방법론으로 도구를 새로 만들 수밖에 없었다 — 편향을 보정하는 정공법이 "소수집단 단독 표본 재추출"임을 두 사례가 공유.
- **SORAG의 PPG(음경측정기) ↔ 신체 측정의 윤리**: 신체 반응을 위험 변수로 환원하는 것은 [[실증주의-범죄학|롬브로조의 신체결정론]]의 유산 — 21세기 정확한 통계와 19세기 골상학의 거리는 생각보다 가깝다. "측정 가능한 것"의 윤리적 한계
- **LS/CMI 캐나다 전산화 ↔ 한국 4세대 공백**: Andrews & Bonta가 RNR 이론을 만들고 LS/CMI로 이론을 도구화·전산화한 것은 "이론 → 측정 → 처우 통합 → 사례관리"의 완결 사이클. 한국은 이 사이클의 중간 고리(4세대) 없이 5세대로 점프 → **이론 적용의 단절** 위험
- **OASys 면담 매뉴얼 비공개 ↔ 평가 도구의 블랙박스**: OASys 내부 질문 매뉴얼이 비공개라는 점 자체가 알고리즘 형사사법의 투명성 문제. 5세대 머신러닝과 동일한 "설명 가능성(explainability)" 쟁점이 3·4세대 구조화 면담 매뉴얼에서도 이미 존재했음 — 인간 평가자도 블랙박스가 될 수 있다
- **RNR 위험성 원칙 ↔ [[교정처우론|마약사범 재활전담 시설의 creaming 효과]]**: 한국 마약사범 재활전담 교정시설(2023~)은 *재활 의지가 강한 초·재범* 위주로 선별 — 고위험 누범자는 배제. 이는 RNR의 *위험성 원칙*(고위험군 집중 개입)과 **정반대 방향**. *효과성 입증이 쉬운 표본*을 골라 정책 가시성을 확보하려는 압력이 RNR 원칙을 압도하는 한국 교정의 구조적 모순. 평가 도구가 아무리 정교해져도 *처우 대상 선별 단계*에서 RNR이 역행되면 5범 이상 59.9% 재복역률의 진짜 표적층은 영구히 프로그램 밖에 남는다.
