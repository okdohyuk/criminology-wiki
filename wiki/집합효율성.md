# 집합효율성 (Collective Efficacy)

> **학습 takeaway**: [[샘슨]]이 [[사회해체이론]]을 현대적으로 재정의한 핵심 개념. "**친한 동네**가 아니라 **실제로 움직이는 동네**"가 범죄를 줄인다. [[깨진-유리창-이론]]의 인과 문제를 우회하면서 공동체 통제의 본질을 더 정확히 짚는다. **전제 조건**은 [[사회자본]] — 콜먼(Coleman)이 말한 "관계망 자체가 자원"이 먼저 있어야 공동행동으로 이어진다.

## 정의

**공공장소에서 질서를 유지할 수 있는 지역사회의 능력**. 두 요소로 구성된다.

1. **사회적 결속과 상호신뢰** — 주민 간 유대와 신뢰
2. **개입하려는 공유된 기대** — 문제가 생겼을 때 실제로 개입할 것이라는 공동의 기대

핵심은 **친밀함** 그 자체가 아니라 **행동으로 이어지는 공유된 기대**다.

## 측정 방법

샘슨의 대표 설문 문항:
- 무단결석하는 학생을 보면 주민들이 나설 것인가?
- 아이들이 낙서하는 것을 보면 개입할 것인가?
- 어른에게 무례한 청소년을 야단치는 사람이 있을 것인가?
- 동네에 예산 삭감이 닥치면 주민들이 함께 항의할 것인가?

**"Yes"가 많을수록 집합효율성이 높다.**

## 사회자본과의 관계 — 콜먼(Coleman)

**집합효율성의 전제 조건**은 [[사회자본]](콜먼, 1988)이다:

- 사회자본 = **관계망 자체가 자원**. 합리적 행위자는 관계가 이익을 가져올 때만 유지한다.
- 이웃 간 신뢰(사회자본)가 쌓여야 → "문제가 생기면 나설 것"이라는 공유 기대(집합효율성)로 이어진다.
- **빈곤·주거불안정 → 사회자본 파괴 → 집합효율성 붕괴 → 범죄율 상승**이 전체 인과 경로.

## 사회해체와의 관계

- **사회해체** = 공동가치를 실현할 능력의 **상실**
- **집합효율성** = 그 능력의 **실현**

빈곤 자체보다 **빈곤 + 주거불안정성**의 결합이 집합효율성을 낮추고, 이것이 범죄율을 올린다.

## 샘슨의 시카고 연구

- **규모**: 시카고 196개 지역 · 12,000블록 조사 ("블랙박스 연구")
- **결과**: 소득·인종·인구 구성이 같아도 **집합효율성이 높은 동네**는 폭력 범죄율이 낮음 → 구조변수보다 집합효율성이 독립적 설명력 가짐
- **추가 발견** (생선 & 그로브): 청소년 비행이 반복되면 성인 범죄로 이어지는 경로에서 지역 집합효율성이 완충 역할

## 박연재(2025) — 집합효율성의 양면적 조절효과 (실증)

**논문**: 박연재 (2025). "무질서와 두려움 사이: 집합효율성의 양면적 조절효과"

### 연구 설계

- **데이터**: 「2020 전국범죄피해조사」(한국형사법무정책연구원), 전국 17개 시도·만 14세 이상 **13,836명** (복합표본 가중치 반영)
- **변수 측정** (모두 5점 척도):
  - 종속변수 — 범죄두려움: 13개 문항, α = .925
  - 독립변수 — 사회적 무질서: 6개 문항, α = .856
  - 조절변수 — 집합효율성: 7개 문항, α = .858
  - 통제변수: 성별·연령·혼인상태·피해경험·학력
- **분석**: 다중회귀(Model 1) + 상호작용항 투입(Model 2) + 제곱항 비선형 모형, VIF 다중공선성 검토

### 주요 회귀계수 (발표 수치)

| 변수 | Model 1 (B) | Model 2 (B) | 해석 |
|------|------------|------------|------|
| 사회적 무질서 | **.955*** | .972*** | 무질서↑ → 두려움↑ (H1 지지) |
| 집합효율성 | **.103*** | .103*** | 집합효율성↑ → 두려움↑ (**역설**, 단순 완화요인 아님) |
| 상호작용항(무질서×집합효율성) | — | **.037*** | 집합효율성이 높을수록 무질서의 영향 **오히려 강화** (H2 방향 역전) |
| 범죄피해 | 1.294*** | .335*** | |
| 성별 | 3.828*** | 3.879*** | |
| R² | .248 | **.252** | ΔR² = .004, ΔF = 73.950*** |

> 상관계수 단계: 범죄두려움 ↔ 무질서 r = .408***, 범죄두려움 ↔ 집합효율성 r = .029***(유의하나 매우 약함), 무질서 ↔ 집합효율성 r = .005(무관) — 상관만 보면 이론이 맞지 않는 것처럼 보이지만, 회귀에서 통제·상호작용·제곱항을 투입하니 비선형성이 드러남.

### 비선형 구조 — 핵심 발견 (제곱항)

- **집합효율성 제곱항 B = −.012 (p < .001)** → **역U형 곡선**
- 중심화 기준 2.68 / 가중치 기준 **23.25점**
- 비선형 모형의 AIC·BIC 모두 감소 → 선형 모형보다 적합도 우수

```
집합효율성 수준:  낮음 ────── 임계점(23.25) ────── 높음
해당 구간:        하위 68%        전환점         상위 32%
효과:             두려움 증가(증폭 구간)        두려움 감소(완화 구간)
```

- **하위 68%**: 집합효율성이 오히려 두려움 **증폭기**로 작동
- **상위 32% (23.25점 초과)**: 비로소 두려움 **완화** 효과 발생
- 함의: 실질적 보호 효과는 **소수 공동체에서만 작동**

### 양면성의 이론적 근거 — 선행연구 (PDF 정리)

| 연구 | 메커니즘 |
|------|---------|
| **Markowitz et al. (2001)** 핵심 명제 | 동일 수준의 무질서라도 집합효율성이 높은 지역에서는 두려움이 낮을 수 있음 — 단, 비공식적 통제력이 *유지될 때* 한정 |
| Thomas (2007) | 재해 후 폐쇄적 공동체 → 소문·불안 빠르게 확산 → 범죄두려움↑ |
| Villarreal & Silva (2006) | 사회적 결속↑ → 위험 정보 활발히 공유 → 위험 인식 *간접* 증대 |
| Goncalo et al. (2010) | 결속 과잉 → 집단사고(Groupthink) → 외부 폐쇄성·의심 강화 |
| Weisburd et al. (2020) | 범죄 Hot Spots 주민참여 개입 → 오히려 범죄 민감도·불안 증가 |
| 김수민 (2024) | 집합효율성 조절효과 유의하나 두려움 *강화* 방향으로 나타날 수 있음 |

→ **양면성의 핵심 메커니즘**: 결속이 위험 정보 공유 채널로 작동 → 임계점 이하에서는 "위험 인식 증폭기"로 기능, 임계점 이상에서야 "비공식 통제력 → 심리적 안정"으로 전환.

### "단톡방 공포" — 증폭 메커니즘

> 아파트 단톡방에 "요즘 무슨 일 있었대"가 올라오면 → 확인되지 않은 소문 공유 → 실제 위험보다 공포 과도하게 증폭

어설픈 집합효율성(낮은 수준의 지역 연대) = 범죄 정보를 과도하게 공유하는 채널로 작동 → 실제 위험보다 공포가 더 커짐.

### 정책 함의

**역효과 가능 정책**: 벽화마을, 단순 주민 결속 캠페인 → 하위 68% 구간에서는 두려움 증폭 유발  
**효과적 방향**: 가로등·CCTV 등 **물리적 환경 개선** 우선, 디지털 기술 활용(사기 탐지 AI 등)

> **핵심 메시지**: 집합효율성은 **양(量)적 확산이 아닌 질(質)적 심화**가 핵심. 단순 유대감 조성을 넘어 **임계수준 이상의 결속·통제력 확보**를 목표로 단계적으로 접근해야 한다.

### 임계값 기반 단계적 개입 전략 (3단계)

| 단계 | 조건 | 정책 패키지 |
|------|------|------------|
| **1단계 — 안정화** | CE < 임계값 (하위 68%) | 물리적 환경 정비(조명·청결) / 경찰 가시성 강화 / 익명 신고 채널 구축 — 결속을 강제하지 말 것 |
| **2단계 — 성장** | CE ≈ 임계값 | 주민 참여 프로그램 도입 / 공동체 규범 형성 워크숍 / 정보공유 플랫폼 구축 |
| **3단계 — 유지** | CE ≥ 임계값 (상위 32%) | 자생적 비공식 통제 강화 / 지역 리더십 발굴·지원 / 성과 측정 및 피드백 |

→ "있다/없다"의 이분법적 정책 → **단계별 맞춤 정책**으로 패러다임 전환 필요.

### 발표자 확장 시각 — 스마트시티 거버넌스

발표자(오지원)의 확장 제언: 주관적 설문 + 객관적 행정·센서 데이터를 결합한 **혼합방법론(Mixed Method)**.

- **CCTV·IoT 센서 데이터**로 무질서 수준 객관화
- **SNS·커뮤니티 앱 활동량**으로 집합효율성 *간접* 측정 → "디지털 집합효율성 지표"
- **신고·민원 데이터**로 비공식 사회통제 대리변수 구성
- **스마트시티 경보 모듈**: 집합효율성 점수 < 임계값 지역은 자동으로 PHASE_1 개입 권장

→ **온라인 공동체로의 확장**: 아파트 단톡방·맘카페·당근마켓·중고나라가 사실상 새로운 집합효율성 플랫폼. **NLP 기반 루머 탐지**로 범죄두려움 선제 관리 가능. 사이버 무질서에 대한 경찰 대응 체계 재설계 필요 → [[AI-매개범죄]] · [[사이버범죄]]와의 정책 접점.

## 설명하는 현상

- 청소년 비행 통제 실패 → 지역 전체 범죄율 상승
- 공공장소 폭력·강도·살인 같은 **중범죄**까지 설명 가능
- 같은 빈곤 수준의 두 동네에서 왜 범죄율이 다른가

> **의외의 연결점**: 사회적 무질서(노숙인 집단, 구걸)를 [[깨진-유리창-이론]]은 "범죄의 원인"으로 읽고, 집합효율성은 "공동체 통제력 부재의 결과"로 읽는다 — 같은 현상을 인과 방향만 바꿔 읽는 두 이론의 역설.

> **의외의 연결점 — [[CPTED]] 민주적 절차와의 동형 구조**: 부산 감천문화마을(주민 참여형) vs 서울 낙산공원·이화마을(시 일방 추진 → 벽화 회색 페인트 사건) 대조는 곧 *집합효율성의 작동 여부* 차이. 같은 물리 설계라도 **주민의 비공식 통제·신뢰가 함께 작동**해야 지속 가능하다는 점에서, 2세대 CPTED(지역사회 협력)는 사실상 집합효율성을 환경설계의 전제로 끌어들인 것 — Sampson의 통찰이 도시설계 정책에 흡수된 사례.

> **의외의 연결점 — 경찰 가시성 U자형과의 동형 구조 ([[범죄에-대한-두려움]])**: 박연재(2025)의 *집합효율성 양면적 조절효과*(임계 이하에서는 두려움 증폭, 이상에서는 완화)는 PDF 30번 슬라이드의 *"경찰 가시성 = 샐러드의 마요네즈"* U자형 비유와 정확히 같은 비선형 구조다. 비공식 통제(집합효율성)와 공식 통제(경찰 가시성) 모두 **임계 모형**을 따른다 — 너무 적으면 보호 부재, 너무 많으면 "위험한 동네" 신호. 통제의 가시성이 *적정 임계* 안에서만 두려움을 낮춘다는 정책적 동형성.
