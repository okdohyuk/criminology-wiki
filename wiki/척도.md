# 척도 (Scale of Measurement)

> **학습 takeaway**: **명목·서열·등간·비율** 4가지 암기 공식을 한 번에 잡자 → `분류 / +순서 / +간격 / +절대영점`. 이걸 외우면 어떤 통계를 써야 하는지 자동으로 결정된다. 성별을 1, 2로 코딩해서 평균을 내면 "반은 남자 반은 여자"라는 **말도 안 되는 해석**이 나오는 이유가 여기 있다.

## 4가지 척도

| 척도 | 속성 | 예시 |
|------|------|------|
| **명목척도** | 분류만 | 성별, 범죄유형, 지역유형 |
| **서열척도** | 분류 + 순서 | 상·중·하, 순위, 만족도 |
| **등간척도** | 분류 + 순서 + 간격 | 온도, IQ, 리커트 점수 |
| **비율척도** | 분류 + 순서 + 간격 + 절대영점 | 키, 몸무게, 소득, 범죄건수 |

## 암기 공식

```
명목 = 분류
서열 = 분류 + 순서
등간 = 분류 + 순서 + 간격
비율 = 분류 + 순서 + 간격 + 절대영점
```

## 절대영점의 의미

- **등간척도의 0**: 속성의 완전한 부재가 아님 (예: 온도 0℃는 "추움 없음"이 아님)
- **비율척도의 0**: 속성의 완전한 부재 (예: 몸무게 0 = 질량 없음)

비율척도만 **"A는 B의 2배"** 같은 비율 비교가 가능.

## 실수 경고

- **성별(1=여자, 2=남자)의 평균 = 1.5** → ❌ "반은 남자"라는 해석 불가
- **명목척도는 이름표일 뿐, 수학 연산의 대상이 아니다**

## 지수(Index) vs 척도(Scale): 헷갈리는 구분

사회과학에서 "척도"는 두 가지 다른 의미로 쓰인다.

| 구분 | 의미 | 문항 간 처리 | 예시 |
|------|------|------------|------|
| **지수 (Index)** | 여러 문항을 **단순 합산**하여 하나의 점수 구성 | 모든 문항 동일 취급 (가중치 없음) | 청소년 비행지수: 무단결석(1)+흡연(1)+절도(1)=3 |
| **척도 (Scale)** | 문항 간 **강도·수준·순서를 고려**하여 점수 부여 후 결합 | 문항마다 차별적 가중 또는 단계적 구조 | 비행심각도척도: 무단결석(1)+흡연(2)+절도(5)=8 |

**같은 행동이라도** 지수로 계산하면 3점, 척도로 계산하면 8점 — 어떤 방식을 쓸지는 연구자의 이론적 판단이다.

> 실무 선택 기준: 문항 간 심각도·강도 차이가 이론적으로 중요하면 척도, 단순히 경험 여부를 합산하면 충분하면 지수.

## 리커트 척도와 연구 실무

사회과학 설문의 대부분은 **리커트형 등간척도**(5점 또는 7점):
- 전혀 그렇지 않다 / 그렇지 않다 / 보통 / 그렇다 / 매우 그렇다

이를 숫자 1~5로 코딩하여 등간척도로 취급한다.
