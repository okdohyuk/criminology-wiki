# 하위문화이론 (Subcultural Theory)

> **학습 takeaway**: 하위문화이론은 [[머튼-아노미이론]]의 "개인 적응"을 **"집단 적응"으로 진화**시킨 이론이다. 긴장에 처한 개인이 혼자 대응하면 도피·혁신이지만, **같은 처지의 집단이 모이면 새로운 문화를 만든다** — 이것이 하위문화. 코헨은 갱이 왜 비실리적으로 보이는지를, 클라워드-올린은 왜 갱의 종류가 다른지를, 밀러는 왜 비행이 반드시 반항이 아닌지를 설명한다. 세 이론의 공통 전제: **비행 하위문화는 합리적(또는 적응적) 집합 행동이다.**

## 이론 가족

```
머튼 아노미이론
    ↓ (집단적 적응으로 확장)
├── 코헨 — 지위문제론 (비행하위문화의 형성)
├── 클라워드·올린 — 차별기회이론 (비행하위문화의 유형)
└── 밀러 — 하층 고유문화론 (비행하위문화의 자율성)
```

## 1. 코헨의 하위문화이론

### 핵심: 지위문제
- 하층 청소년은 중간계급 기준의 학교·사회에서 **지위 달성에 실패**
- 동일한 문제를 겪는 하층 청소년들이 서로 상호작용 → **새 문화 형성** (= "문화적 혁신")
- 갱·또래집단을 **준거집단(reference group)**으로 삼아 지위문제를 **집합적으로 해결**

### 학교: 중간계급 문화의 재생산 공간

학교는 겉으로는 모든 학생에게 열려 있지만, 실제로는 **중간계급의 언어·태도·행동 양식**을 정상 기준으로 삼는 공간이다.

**계급별 언어 불평등 사례**:
- 영국: 화장실을 어떻게 부르느냐만 들어도 계급을 알 수 있음
  - 상층: *Restroom* / 중간층: *Toilet* / 하층: *Loo (Roo)*
- 미국: 인종·민족 집단별 슬랭 체계가 달라 학교에서의 "정상" 언어는 곧 백인 중산층 언어

→ 하층 청소년은 이 기준을 갖추지 못한 채 학교에 진입 → 교사와의 관계 악화 → 학업 실패 → 주변화

**결과**: 학교 안에서 반복적으로 지위 획득에 실패한 청소년들이 **비슷한 처지의 또래와 상호작용** → 스스로 새 기준을 세우고 그 안에서 지위를 획득하는 **대안적 문화** 형성.

코헨은 이를 **"문화적 혁신(cultural innovation)"**이라 불렀다 — 기존 지배문화를 더 이상 기준으로 삼지 않고 집단 자체의 새 가치 체계를 수립하는 것.

### 비행하위문화의 특성
- **비실리적(non-utilitarian)**: 직접적 이득 없이 파괴 — 경제적 동기가 없는 비행
- **악의적(malicious)**: 타인에 대한 의도적 반감 표출
- **부정적(negativistic)**: 지배규범을 의도적으로 거꾸로 뒤집음

→ 비행 하위문화는 단순한 일탈 모임이 아니라, 주류에서 지위를 얻지 못한 청소년들이 **집합적으로 만들어낸 대안적 질서**다.

### 악순환 구조
```
지위 달성 실패 → 갱 가입(내부 지위 획득) → 사회 전체에서 지위 더 낮아짐
→ 갱 연대 강화 → 지배문화와 괴리 심화 → 비행하위문화 공고화
```

## 2. 클라워드·올린의 차별기회이론 → [[클라워드-올린-차별기회이론]]

- 머튼의 한계 비판: "불법적 수단도 누구나 접근할 수 있는 게 아니다"
- **합법적 수단 + 불법적 수단** 모두 차별적으로 분배

### 비행하위문화 3유형

| 유형 | 특징 | 등장 조건 |
|------|------|----------|
| **범죄 하위문화** | 조직화된 범죄, 경제적 성공 지향 | 성인 범죄자와의 연결·역할 모형 존재 |
| **갈등 하위문화** | 폭력, 지위를 폭력으로 획득 | 합법적·불법적 기회 모두 막힘, 폭력만 가능 |
| **도피 하위문화** | 약물, 빈둥거림 | "이중의 실패" — 합법·불법 모든 기회 차단 |

## 3. 밀러의 하층계급문화론

- 코헨에 반박: 비행하위문화는 중산층 문화와의 갈등에서 발생한 **반대문화가 아니라**, **하층계급 자체의 고유문화**
- 오랫동안 존속하며 자체적 통합력을 가진 **유형화된 전통**
- 이 가치에 충실하게 사는 것 = 자연스럽게 비행 → **비행은 반항이 아니라 적응**

### 6가지 관심의 초점 (Focal Concerns)

| 관심 초점 | 의미 |
|------|------|
| **말썽(Trouble)** | 중산층이 업적을 평가 기준으로 삼는다면, 하층은 **권위에 분쟁거리를 만들어내고 잘 빠져나오는 것**이 평가 기준 |
| **강인함(Toughness)** | 힘, 용감성, 대담성, 거침, 남자다움 |
| **교활함(Smartness)** | 남을 잘 속이는 반면 자신은 남에게 속지 않으려는 능력 |
| **자극(Excitement)** | 스릴이나 모험심 등 권태감을 해소하려는 관심 |
| **운명주의(Fate)** | 운수·재수가 자기 인생을 좌우한다는 관점 — 외적 요소가 결정적 |
| **자율성(Autonomy)** | 일종의 방종 — 다른 사람의 간섭을 받기 싫어함 |

→ 6가지 관심초점에 충실하면 **자동으로 형법과 충돌**한다. 따라서 비행은 "반항의 결과"가 아니라 **하층계급 사회화의 자연스러운 산물**.

## 형사사법 정책 함의

하위문화이론은 두 갈래의 상충하는 정책 함의를 동시에 낳는다.

### A. 갱·하위문화 해체 접근

- 갱 가입 청소년 분리, 또래집단 재구성
- 학교 내 멘토링, 다른 준거집단(스포츠·문화 동아리) 제공
- 미국의 "Big Brothers Big Sisters", 한국의 학교폭력 가해자 분반·대안교실

### B. 기회구조 개입 (클라워드-올린)

- 합법적 기회구조 확대: 직업훈련, 교육 접근성
- 1960년대 미국 "Mobilization for Youth" 프로젝트 → 케네디·존슨 시기 "빈곤과의 전쟁" 정책의 이론적 기반

### C. 하층계급문화 자체에 대한 개입 한계 (밀러의 함의)

- 밀러 이론이 옳다면 비행하위문화는 단순한 일탈집단이 아니라 **하나의 생활양식**
- → 단기 교정개입만으로 변화 어렵고, **세대를 가로지르는 사회구조적 정책**(주거·교육 통합) 필요
- 한국 사례: "위기청소년" 분리·격리 정책의 한계 — 가족·지역사회 단위 통합 개입 필요성

**[[낙인이론]]과의 긴장**: 하위문화 해체 정책은 갱 가입 청소년을 식별·분리하는 과정에서 **새로운 낙인을 만들 위험**. → [[회복적-사법]]과 결합한 비공식적 개입이 대안.

## 비판과 평가

- 비행집단이 **좌절한 청소년에게 나름의 기능**을 함을 보여줌 (낙인만 찍지 말라는 함의)
- 코헨 vs 클라워드-올린 차이: **왜 갱이 다양한가** → 불법적 기회 접근의 차이
- 한계: 여성·중산층 비행 설명 어려움

## 의외의 연결점

- **코헨의 지위 박탈 ↔ 스탠퍼드 감옥 실험**: 코헨은 학교에서 지위를 얻지 못한 청소년이 갱에서 지위를 찾는다고 했다. 스탠퍼드 감옥 실험에서는 역할(간수/죄수)이 부여되자 그 역할에 맞게 행동이 달라졌다. 공통점: **지위와 역할이 행동을 결정한다** — 개인의 본성이 아니라 **구조적 위치**가 행동을 만든다 → [[밀그램-복종-실험]], [[범죄-연구방법]]
- **[[조직범죄]]와 범죄 하위문화**: 클라워드-올린의 '범죄 하위문화'는 성인 조직범죄의 세대 전승 메커니즘 — 갱이 범죄 조직으로 제도화되는 경로
- **[[약물-혐오-스토킹-환경범죄|약물]]과 도피 하위문화**: 클라워드-올린의 "이중의 실패" 청소년 = 약물 범죄 취약층
- **[[사회해체이론]]과 연결**: 쇼와 맥케이의 "비행의 전승" = 하위문화이론의 세대 전승 메커니즘과 정확히 일치
- **[[일반긴장이론]]과 연결**: GST의 "긴장 → 부정감정 → 범죄"가 집단 수준으로 확장되면 하위문화 형성 — 집단의 분노가 문화가 됨
- **[[깨진-유리창-이론]]과 긴장**: 무질서한 환경(깨진 유리창)은 갈등 하위문화의 서식지 → CPTED가 하위문화 생성 자체를 억제할 수 있다는 함의
