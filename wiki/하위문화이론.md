# 하위문화이론 (Subcultural Theory)

> **학습 takeaway**: 하위문화이론은 [[머튼-아노미이론]]의 "개인 적응"을 **"집단 적응"으로 진화**시킨 이론이다. 긴장에 처한 개인이 혼자 대응하면 도피·혁신이지만, **같은 처지의 집단이 모이면 새로운 문화를 만든다** — 이것이 하위문화. 코헨은 갱이 왜 비실리적으로 보이는지를, 클라워드-올린은 왜 갱의 종류가 다른지를, 밀러는 왜 비행이 반드시 반항이 아닌지를 설명한다. 세 이론의 공통 전제: **비행 하위문화는 합리적(또는 적응적) 집합 행동이다.**

## 이론 가족

```
머튼 아노미이론
    ↓ (집단적 적응으로 확장)
├── 코헨 — 지위문제론 (비행하위문화의 형성)
├── 클라워드·올린 — 차별기회이론 (비행하위문화의 유형)
└── 밀러 — 하층 고유문화론 (비행하위문화의 자율성)
```

## 1. 코헨의 하위문화이론

### 핵심: 지위문제
- 하층 청소년은 중간계급 기준의 학교·사회에서 **지위 달성에 실패**
- 동일한 문제를 겪는 하층 청소년들이 서로 상호작용 → **새 문화 형성** (= "문화적 혁신")
- 갱을 준거집단으로 삼아 지위문제를 **집합적으로 해결**

### 비행하위문화의 특성
- **비실리적(non-utilitarian)**: 이득 없이 파괴
- **악의적(malicious)**: 중간계급 가치에 대한 노골적 반대
- **부정적(negativistic)**: 지배규범을 거꾸로 뒤집음

### 악순환 구조
```
지위 달성 실패 → 갱 가입(내부 지위 획득) → 사회 전체에서 지위 더 낮아짐
→ 갱 연대 강화 → 지배문화와 괴리 심화 → 비행하위문화 공고화
```

## 2. 클라워드·올린의 차별기회이론 → [[클라워드-올린-차별기회이론]]

- 머튼의 한계 비판: "불법적 수단도 누구나 접근할 수 있는 게 아니다"
- **합법적 수단 + 불법적 수단** 모두 차별적으로 분배

### 비행하위문화 3유형

| 유형 | 특징 | 등장 조건 |
|------|------|----------|
| **범죄 하위문화** | 조직화된 범죄, 경제적 성공 지향 | 성인 범죄자와의 연결·역할 모형 존재 |
| **갈등 하위문화** | 폭력, 지위를 폭력으로 획득 | 합법적·불법적 기회 모두 막힘, 폭력만 가능 |
| **도피 하위문화** | 약물, 빈둥거림 | "이중의 실패" — 합법·불법 모든 기회 차단 |

## 3. 밀러의 하층 고유문화론

- 코헨에 반박: 비행이 **지배문화에 대한 반동**이 아니라 **하층 고유 문화의 자연스러운 산물**
- 하층계급 문화의 핵심 가치: 말썽(trouble), 거칠고 강인함(toughness), 약삭빠름(smartness), 흥분·짜릿함(excitement)
- 이 가치에 충실하게 사는 것 = 자연스럽게 비행 → **비행은 반항이 아니라 적응**

## 비판과 평가

- 비행집단이 **좌절한 청소년에게 나름의 기능**을 함을 보여줌 (낙인만 찍지 말라는 함의)
- 코헨 vs 클라워드-올린 차이: **왜 갱이 다양한가** → 불법적 기회 접근의 차이
- 한계: 여성·중산층 비행 설명 어려움

## 의외의 연결점

- **[[조직범죄]]와 범죄 하위문화**: 클라워드-올린의 '범죄 하위문화'는 성인 조직범죄의 세대 전승 메커니즘 — 갱이 범죄 조직으로 제도화되는 경로
- **[[약물-혐오-스토킹-환경범죄|약물]]과 도피 하위문화**: 클라워드-올린의 "이중의 실패" 청소년 = 약물 범죄 취약층
- **[[사회해체이론]]과 연결**: 쇼와 맥케이의 "비행의 전승" = 하위문화이론의 세대 전승 메커니즘과 정확히 일치
- **[[일반긴장이론]]과 연결**: GST의 "긴장 → 부정감정 → 범죄"가 집단 수준으로 확장되면 하위문화 형성 — 집단의 분노가 문화가 됨
- **[[깨진-유리창-이론]]과 긴장**: 무질서한 환경(깨진 유리창)은 갈등 하위문화의 서식지 → CPTED가 하위문화 생성 자체를 억제할 수 있다는 함의
