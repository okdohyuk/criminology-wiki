# 화이트칼라범죄 (White-Collar Crime)

> **학습 takeaway**: 서덜랜드가 제창한 개념. **거리의 범죄**보다 덜 폭력적으로 보이지만 **사회 전체의 신뢰를 무너뜨리는 효과는 더 크다**. "이 정도는 업계 관행"이라는 **중화기법**(자기합리화)이 이 범죄의 진짜 엔진. 부패·금융범죄·기업범죄를 읽는 공통 렌즈.

## 정의

**사회적 지위가 높고 직업활동을 수행하는 과정**에서 범해지는 범죄.

## 서덜랜드의 핵심 주장

전통적 거리범죄보다 **덜 폭력적으로 보이지만** 사회 전체의 **신뢰를 무너뜨리는 효과**는 오히려 더 크다.

## 주요 특징

- **적발 어려움** — 전문지식·내부관계 이용
- **피해 분산** — 수많은 사람이 조금씩 피해
- **불법성 모호** — 합법과 위법의 경계에 걸림
- **피해자 인식 지연** — 피해자 스스로 침해를 인식하지 못함

## 구체적 행위

- 회계조작
- 탈세
- 부당광고
- 배임·횡령
- 주가조작
- 입찰담합

## 중화기법(Neutralization)의 역할 — 사이크스(Sykes)

사이크스(Gresham Sykes)가 체계화한 **5가지 중화기법**이 화이트칼라범죄의 논리 엔진이다:

| 중화기법 | 내용 | 화이트칼라 사례 |
|----------|------|--------------|
| **책임 부정** | "내 탓이 아니다, 시스템·상황 때문" | "회사 방침이라 어쩔 수 없었다" |
| **피해 부정** | "실질적 해가 없다" | "분식회계인데 실제 손해 입은 사람 있나?" |
| **피해자 부정** | "저 사람들이 당할 만했다" | "그 투자자들이 욕심 부렸잖아" |
| **비난자 비난** | "너희도 똑같이 한다" | "검찰도 정치적으로 움직이잖아" |
| **높은 충성심 호소** | "더 큰 이익을 위해" | "직원들 먹여 살리려고 어쩔 수 없이" |

> **의외의 연결점**: 중화기법은 [[하위문화이론]]의 **중화기술(Techniques of Neutralization)**과 동일한 사이크스의 이론 — 청소년 비행과 고위층 범죄가 **동일한 자기합리화 메커니즘**으로 작동한다는 것이 핵심 역설.

## 수법별 7가지

1. **허위 자산 등재** — 재무제표 조작 (분식회계)
2. **과대청구** — 실제보다 많은 금액 청구 (보험사기, 정부 조달)
3. **뇌물 제공** — 결정권자에게 금품·향응
4. **내부자 정보** — 비공개 정보로 주식 거래
5. **담합** — 경쟁사와 가격·입찰 조율
6. **투자 사기** — 폰지 구조·다단계 (버나드 매도프 사례)
7. **환경·안전 기준 위반** — 비용 절감을 위한 규제 무시

### 범행동기에 따른 분류

| 대분류 | 소분류 | 예시 |
|--------|--------|------|
| **조직적 범죄** | 사기·기만형 | 사기, 부당광고, 탈세 |
| | 시장통제형 | 불공정거래, 가격담합 |
| | 뇌물매수형(상업적) | 담당공무원에게 계약 수주 대가 제공 |
| | 뇌물매수형(정치적) | 불법 정치자금 제공 |
| | 기본권침해형 | 권력을 이용한 인간 기본권 침해 |
| **직업적 범죄** | 기업범죄 | 분식회계, 주가조작, 횡령·배임 |
| | 정부범죄 | 공무원 직무유기·직권남용·수뢰 |
| | 전문가범죄 | 의사·변호사·회계사의 직업윤리 위반 |

> **의외의 연결점**: 정부범죄(공무원 범죄)는 [[정치범죄]]의 "정부에 의한 범죄"와 겹친다 — 권력이 범죄를 정의하는 동시에 권력 자체가 범죄를 저지르는 역설
