# 화이트칼라범죄 (White-Collar Crime)

> **학습 takeaway**: 서덜랜드가 제창한 개념. **거리의 범죄**보다 덜 폭력적으로 보이지만 **사회 전체의 신뢰를 무너뜨리는 효과는 더 크다**. "이 정도는 업계 관행"이라는 **중화기법**(자기합리화)이 이 범죄의 진짜 엔진. 부패·금융범죄·기업범죄를 읽는 공통 렌즈.

## 정의

**사회적 지위가 높고 직업활동을 수행하는 과정**에서 범해지는 범죄.

## 서덜랜드의 핵심 주장

전통적 거리범죄보다 **덜 폭력적으로 보이지만** 사회 전체의 **신뢰를 무너뜨리는 효과**는 오히려 더 크다.

## 주요 특징

- **적발 어려움** — 전문지식·내부관계 이용
- **피해 분산** — 수많은 사람이 조금씩 피해
- **불법성 모호** — 합법과 위법의 경계에 걸림
- **피해자 인식 지연** — 피해자 스스로 침해를 인식하지 못함

## 구체적 행위

- 회계조작
- 탈세
- 부당광고
- 배임·횡령
- 주가조작
- 입찰담합

## 중화기법(Neutralization)의 역할

화이트칼라범죄는 **자기합리화**로 자라난다:

- "이 정도는 업계 관행"
- "다들 하는데 나만 안 하면 손해"
- "큰 회사를 상대로 하는 건 괜찮다"
- "불법주차 남들도 다 하니까 잠깐만"

일상에서 흔히 쓰는 변명이 확장된 것이 화이트칼라범죄의 논리적 토대.

## 유형 분류

### 범행동기에 따른 분류

| 대분류 | 소분류 | 예시 |
|--------|--------|------|
| **조직적 범죄** | 사기·기만형 | 사기, 부당광고, 탈세 |
| | 시장통제형 | 불공정거래, 가격담합 |
| | 뇌물매수형(상업적) | 담당공무원에게 계약 수주 대가 제공 |
| | 뇌물매수형(정치적) | 불법 정치자금 제공 |
| | 기본권침해형 | 권력을 이용한 인간 기본권 침해 |
| **직업적 범죄** | 기업범죄 | 분식회계, 주가조작, 횡령·배임 |
| | 정부범죄 | 공무원 직무유기·직권남용·수뢰 |
| | 전문가범죄 | 의사·변호사·회계사의 직업윤리 위반 |

> **의외의 연결점**: 정부범죄(공무원 범죄)는 [[정치범죄]]의 "정부에 의한 범죄"와 겹친다 — 권력이 범죄를 정의하는 동시에 권력 자체가 범죄를 저지르는 역설
